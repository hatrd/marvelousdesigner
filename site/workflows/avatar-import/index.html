    头像导入指南¶

    指南信息

    预计用时: 约 20 分钟 | 难度: 适合初学者 | 重要度: 必做

    本页目标: 将你的 VRChat 头像（FBX 文件）导入 Marvelous Designer，并完成服装制作前的准备。

    开始前准备

    • 已理解 Marvelous Designer 基础界面
    • 已准备好 VRChat 头像的 FBX 文件
    • Marvelous Designer 可以正常运行

    🎯 本次要达成的目标¶

    1. ✅ 正确将 FBX 文件导入 Marvelous Designer
    2. ✅ 合理设置头像的尺寸和姿势
    3. ✅ 调整到适合制作服装的状态
    4. ✅ 理解导入设置，方便之后顺畅重复操作

    📁 文件准备¶

    步骤 1: 确认 FBX 文件¶

    步骤 1-1: 提前检查文件

    导入头像前，先确认 FBX 文件的状态。

    检查项目:

    • ✅ 文件大小: 1MB 到 50MB 左右属于正常范围
    • ✅ 文件名: 仅使用英数字，避免日文或中文
    • ✅ 保存位置: 放在 VRChat衣装制作/Avatars/ 文件夹中
    • ✅ 文件完整性: 文件未损坏

    理想的文件名示例: - ✅ MyAvatar_v1.fbx - ✅ avatar_female_base.fbx - ❌ 私のアバター.fbx （日文文件名） - ❌ avatar file (new).fbx （包含空格和括号）

    关于 FBX 文件

    直接使用 VRChat 里使用的头像文件即可，不需要额外转换或特殊准备。

    🚀 头像导入步骤¶

    步骤 2: 在 Marvelous Designer 中导入¶

    步骤 2-1: 执行导入

    1. 启动 Marvelous Designer，打开一个新项目
    2. 在顶部菜单点击 “Avatar” → “Load Avatar”
    3. 打开文件选择对话框
    4. 选择你的 FBX 文件后点击 “打开”

    开始导入后: - 会显示进度条 - 处理通常需要 30 秒到 2 分钟 - 请等待直到处理完成

    导入过程中的注意事项

    导入时不要执行其他操作，否则可能中断处理流程。

    步骤 3: 确认导入结果¶

    步骤 3-1: 检查头像显示

    导入完成后，确认 3D 视口：

    正常导入时应看到: - ✅ 头像显示在 3D 视口中 - ✅ 以 T 姿势（双臂水平展开）显示 - ✅ 整个头像位于画面内 - ✅ 纹理（服装颜色、皮肤颜色）显示正常

    显示调整方法: 1. 按 “F” 键让头像适配到画面中 2. 按住鼠标右键拖动，从不同角度检查头像 3. 按 “2” 键切换到着色显示

    导入成功的判断标准

    如果头像显示正常，并且可以辨认脸部和身体特征，就说明导入成功。

    步骤 4: 调整头像设置¶

    步骤 4-1: 检查基础设置

    在 3D Object Browser 中检查并调整头像设置：

    在 Object Browser 中确认: 1. 查看右下方的 3D Object Browser 2. 在 “Avatar” 区块中确认已显示头像名称 3. 点击头像名称，使其处于选中状态

    在 Property Editor 中可调整的项目: - Scale: 头像尺寸（通常保持 1.0 即可） - Show Avatar: 切换显示/隐藏 - Avatar Opacity: 透明度（需要透视观察服装时可调整）

    步骤 4-2: 调整姿势（可选）

    如有需要，也可以调整头像姿势：

    修改姿势的步骤: 1. 点击 “Avatar” 菜单 → “Edit Avatar” 2. 选择 “Pose” 标签页 3. 从预设姿势中选择： - T-Pose: 基础姿势（推荐） - A-Pose: 双臂稍微下放的姿势 - Natural: 自然站立姿势

    进行服装制作时，T-Pose 最合适

    🔧 故障排查¶

    常见问题与解决方法¶

    “无法导入 FBX 文件”

    原因与处理方法:

    1. 文件路径问题:
    2. 确认文件路径中不包含日文或中文

    3. 将文件移动到更短的路径位置

    4. 文件损坏:

    5. 用其他软件（如 Blender）确认 FBX 文件能否正常打开

    6. 重新下载或重新导出原始文件

    7. 版本兼容性:

    8. 如果 FBX 文件版本过旧，可能需要重新导出
    “头像不显示 / 看不见”

    处理方法:

    1. 检查显示设置:
    2. 在 Object Browser 中确认头像的 👁️ 图标是否已启用

    3. 在 Property Editor 中确认 Avatar Opacity 不是 0

    4. 调整相机位置:

    5. 按 “F” 键聚焦到头像

    6. 使用鼠标滚轮缩小视图

    7. 切换显示模式:

    8. 按 “2” 键切换到着色显示
    9. 按 “3” 键切换到纹理显示
    “头像太大 / 太小”

    处理方法:

    1. 调整缩放:
    2. 在 Property Editor 中调整 Scale 数值

    3. VRChat 头像通常在 0.8 到 1.2 之间

    4. 确认合适尺寸:

    5. 调整到头像身高约为 1.5m 到 2.0m
    6. 同时考虑后续服装制作的操作便利性
    “纹理（颜色 / 花纹）不显示”

    处理方法:

    1. 切换显示模式:
    2. 按 “3” 键切换到纹理显示模式

    3. 按 “4” 键切换到渲染显示模式

    4. 检查文件结构:

    5. 确认纹理文件与 FBX 文件位于相同位置
    6. 确认该 FBX 是否为内嵌纹理的文件

    关于纹理

    这不会影响服装制作的基础流程，但在最终确认时会很重要。

    ✅ 完成检查清单¶

    完成以下全部项目后，就表示头像导入成功：

    头像导入完成检查

    • 已成功导入 FBX 文件
    • 头像已显示在 3D 视口中
    • 已正确以 T 姿势显示
    • 按 “F” 键后头像整体能完整显示在画面中
    • Object Browser 中已显示 Avatar 项目
    • 可以进行基本视角操作（旋转、缩放）
    • 可以辨认头像的主要特征（脸部、体型）
    • 已保存为项目文件

    📚 参考信息¶

    关于头像文件¶

    FBX 文件中包含的信息: - 网格: 3D 形状数据 - 骨骼: 骨架信息（用于动画） - 纹理: 颜色与花纹信息 - 材质: 材质信息

    VRChat 头像的特点: - 已针对 Unity 进行优化 - 骨骼布局为 Humanoid 形式 - 纹理符合 VRChat 的使用方式

    项目管理技巧¶

    推荐的文件整理方式: 

    VRChat衣装制作/
├── Avatars/
│   ├── MyAvatar_v1.fbx
│   └── MyAvatar_textures/ (有纹理时)
├── Projects/
│   └── MyAvatar_dress_project.zprj
└── Exports/
    └── (完成的服装文件)

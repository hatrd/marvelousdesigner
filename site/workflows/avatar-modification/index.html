    🎯 改造作业全貌¶

    改造流程的 4 个阶段¶

    1. 📋 分析阶段: 分析头像结构并找出问题
    2. 🔧 准备阶段: 制作适配套装
    3. ⚡ 自动化阶段: 执行 Auto Fitting 功能
    4. 🎨 完成阶段: 手动调整并导出供 VRChat 使用

    大概要花多久？¶

    作业时间参考（现在更简单了！）

    👤 普通人形头像 🕐 约 30 - 45 分钟 完成（非常快！） - 自动处理: 5 - 15 分钟（由 AI 完成） - 微调: 15 - 30 分钟（整理细节部分）

    🎭 稍微特殊的体型 🕐 约 45 - 85 分钟 完成 - 自动处理: 10 - 25 分钟 - 微调: 30 - 60 分钟（手动部分会稍微增加）

    🐱 动物系 / 非人形 🕐 约 75 - 125 分钟 完成 - 自动处理: 15 - 35 分钟 - 微调: 60 - 90 分钟（由于不同于人类，需要额外调整）

    🆕 2025 年节省时间的构成

    主要效率提升因素: - Re-target Draping: 适配套装制作时间缩短 90% - Enhanced Auto Fitting: 手动调整流程减少 60% - EveryWear 优化: VRChat 调整作业自动化 80% - AI 辅助: 错误发生率降低 50%，大幅减少返工

    📋 第 1 阶段: 导入并检查头像¶

    步骤 1: 先准备好头像¶

    步骤 1-1: 准备头像文件

    需要的内容: - 想在 VRChat 中使用的头像 FBX 文件 - 想穿上的服装文件（PZIP 格式）

    将头像导入到 Marvelous Designer: 1. 启动 Marvelous Designer 2. 点击 “File” → “Import” → “Avatar” 3. 选择头像的 FBX 文件 4. 点击 “OK” 导入

    检查是否导入成功: - ✅ 成功: 头像会显示在 3D 视图中 - ⚠️ 小问题: 虽然显示了，但看起来有点奇怪 - ❌ 失败: 出现错误，或什么都没显示

    💡 给新手的建议

    一开始只要能“成功”导入就可以！即使有些小问题，大多也能在后面修好。

    非兼容头像常见的特有问题

    • 骨骼名称与标准不一致
    • 缩放差异很大
    • 拥有特殊的网格结构
    • 纹理无法正确显示

    步骤 2: 检查头像体型¶

    步骤 2-1: 这是什么样的体型？

    需要观察的要点: - 大小: 大？小？普通？ - 体型: 偏瘦？丰满？肌肉型？ - 特殊部位: 有尾巴、翅膀、角之类吗？

    简单检查方法: 1. 在 3D 视图里把头像转一圈看整体 2. 记下它和普通人类相比有哪些不同 3. 确认是否有特别明显不同的部位

    常见类型: - 👤 像人类: 最简单！ - 🎭 是人类但体型特殊: 稍微花点功夫就能做好 - 🐱 动物系: 有挑战，但可以做到 - 🤖 机器人 / 机械: 需要一些技巧

    🔧 第 2 阶段: 制作魔法套装¶

    步骤 3: 什么是适配套装？¶

    步骤 3-1: 魔法套装的真面目

    适配套装 = 能记住体型的套装

    简单来说： - 😊 一套能精确记住头像体型的轻薄套装 - ✨ 有了它，服装就能自动适配对应体型 - 🎯 只要做一次，就能反复用于各种服装

    为什么需要它？: - 让 AI 能理解“这个头像原来是这样的体型” - 能大幅减少手工作业 - 出错概率更低

    💡 一个好懂的比喻

    就像去服装店量尺寸时用的软尺！只要量过一次，以后做任何衣服都能按你的体型来。

    🆕 2025 年新功能: 使用 Re-target Draping

    Re-target Draping（重定向垂坠塑形） 是 2025 年的革新功能：

    主要优点: - 完整保留 UV 映射和纹理 - 自动将现有服装数据调整到不同体型 - 大幅缩短手动调整时间（较以往减少 70%） - 自动实现专业级贴合效果

    最适合使用的情况: - 体型差异很大的非兼容头像（尺寸差超过 +30%） - 想利用带有高质量纹理的现有服装 - 需要把同一套服装适配到多个头像时

    步骤 4: 开始制作适配套装¶

    步骤 4-1: 🆕 2025 年推荐: 优先使用 Re-target Draping 的方法

    新的 2025 工作流（推荐）: 1. 先导入现有服装文件（PZIP） 2. 选择 “Avatar” → “Re-target Draping” 3. 确认自动检测出的 “Source Avatar”（原服装使用的头像） 4. 选择 “Target Avatar”（新的非兼容头像） 5. “Quality Level”: 选择 High（高质量，推荐） 6. “Preserve UV Mapping”: 必须勾选 7. “Auto Create Fitting Suit”: 勾选（自动生成） 8. 点击 “Execute Re-targeting” 执行

    Re-target Draping 执行结果: - 会自动生成适配套装 - 服装会自动适配到新头像 - 纹理和 UV 映射会被完整保留 - 处理时间: 2 - 5 分钟（相当于过去 4 - 8 小时的手工作业）

    步骤 4-2: 传统方式（不支持 Re-target 时）

    传统的自动创建方式: 1. 确认当前处于选中头像的状态 2. 点击 “Avatar” → “Create Fitting Suit” 3. 勾选 “Auto Reference Line” 4. 点击 “OK” 执行自动创建

    确认自动创建成功: - 头像周围会生成适配套装 - 主要体型点位（胸、腰、臀等）会被正确放置 - 不会显示错误信息

    步骤 4-3: 手动创建（自动方式失败时）

    手动设置参考线: 1. “Avatar” → “Create Fitting Suit” 2. 选择 “Manual Reference Line” 3. 按列表顺序设置参考线： - Chest（胸部）: 胸部最宽位置的 3 个点 - Waist（腰部）: 最细位置的 3 个点 - Hip（臀部）: 最宽位置的 3 个点 4. 为每条参考线准确指定 3 个点

    设置参考线的技巧: - 以前方、侧面、后方 3 个点构成圆形来布置 - 按照头像自然的体型轮廓来设置 - 注意仔细布置，确保左右对称

    ⚡ 第 3 阶段: 让 AI 帮你给衣服上身¶

    步骤 5: 导入服装¶

    步骤 5-1: 把服装拿进来

    导入服装的步骤: 1. 在适配套装已经制作完成的状态下继续 2. 点击 “File” → “Import” → “Garment” 3. 选择想穿上的服装 PZIP 文件 4. 点击 “OK” 导入

    检查是否导入成功: - 左侧 2D 视图会显示服装版型 - 右侧列表会新增服装名称 - 此时还不会变成 3D 服装

    💡 给新手的补充说明

    这个阶段的衣服还只是平面的版型。下一步 AI 会把它自动穿成立体效果！

    Auto Fitting 前的重要设置

    为了让 Auto Fitting 功能正常运行，请务必完成以下设置：

    1. 调整粒子间距: 将 3D 服装的 Particle Distance 设为 20mm 以上
    2. 属性恢复默认: 将 3D 服装的属性改回默认值

    步骤 6: 发动 AI 魔法！¶

    步骤 6-1: 🚀 使用新的 AI 自动适配功能

    AI 会帮你做的事: - 🧠 智能体型识别: 准确理解头像体型 - 🎯 按部位调整: 分别优化胸、腰、臀 - ✨ 自然褶皱: 自动计算真实的布料褶皱 - 👕 支持叠穿: 可以同时处理内层和外层服装

    简单执行步骤: 1. 选择 “Auto Fit” → “Enhanced Mode” 2. 将 “AI-Assisted Fitting” 打开（推荐） 3. 将 “Body Zone Optimization” 打开 4. 将 “Natural Wrinkle Prediction” 设为 “Medium” 5. “Quality vs Speed” 选择“平衡” 6. 点击 “Execute Enhanced Fitting”！

    处理过程中会看到这些内容: - 会实时显示进度 - 能看到当前正在调整哪个部位 - 还会提示预计完成时间 - 约 3 - 8 分钟完成（够喝一杯咖啡的时间）

    步骤 6-2: 🆕 Automatic Fitting Suit Creation（2025.0 新功能）

    不再需要传统适配套装的革新功能:

    什么是 Automatic Fitting Suit Creation: - 从头像自动生成最优的适配套装 - 完全不再需要手动设置参考线 - 自动分析骨骼结构并决定套装形状

    使用条件: - Marvelous Designer 2025.0 及以上 - 支持 Humanoid 绑定骨架的头像（推荐） - 头像的多边形数量: 20K - 100K（最佳范围）

    执行步骤: 1. 导入头像后、导入服装前执行 2. “Avatar” → “Auto Create Fitting Suit” → “Advanced Mode” 3. “Body Type Detection”: Auto（自动检测体型） 4. “Suit Complexity”: Medium（推荐） 5. 点击 “Create & Apply” 6. 自动生成后即可直接导入服装

    步骤 6-3: 结果确认与质量评估

    Enhanced Auto Fitting 完成后的检查: 1. 通过 “Simulation” → “Preview Mode” 预先确认 2. 确认已自动生成 质量报告（2025 年新功能） 3. “Fit Quality Score”: 理想值为 85 分以上（满分 100） 4. 必要时可用 “Re-fit Selected Areas” 对局部重新处理

    预期结果（2025 年大幅提升）: - 对头像实现精确贴合（比过去提高 40%） - 自然的褶皱与布料流动感 - 保证动作时的稳定性 - 自动应用 VRChat 优化参数

    步骤 9: 手动调整技巧¶

    步骤 9-1: 解决局部问题

    常见问题与处理方法:

    胸部适配问题: 

    问题: 胸部位置的服装悬空或嵌入
处理: 手动调整胸部版型的曲线
步骤:
1. 选择胸部版型
2. 调整 Edit Pattern → Curve Point
3. 微调内外侧曲线
4. 使用 Symmetry 功能应用左右对称

    袖长调整: 

    问题: 袖子过长或过短
处理: 调整袖口版型的位置
步骤:
1. 选择袖子版型的下摆部分
2. 通过 Transform → Move 进行垂直移动
3. 将两只袖子调成相同长度
4. 必要时也调整袖口形状

    裙摆 / 下摆调整: 

    问题: 下摆展开得不自然
处理: 调整下摆的曲线与长度
步骤:
1. 选择下摆版型
2. 通过曲线点调整展开程度
3. 通过 Transform → Scale 调整整体尺寸
4. 在 Simulation 中确认结果

    步骤 9-2: 高级调整技巧

    比例调整: 1. “Edit Pattern” → “Advanced Transform” 2. 使用 “Non-uniform Scaling” 分别调整纵横比例 3. 使用 “Proportional Edit” 控制对周边区域的影响

    缝线优化: 1. “2D Pattern” → “Edit Internal Line” 2. 微调缝线的位置与角度 3. 使用 “Sewing Strength” 调整缝合强度

    VRChat 兼容性: - [ ] 文件大小合适（推荐 50MB 以下） - [ ] 导出设置正确 - [ ] 保留了 Unity 集成所需的信息

    步骤 11-1: 面向 VRChat 的导出设置

    导出前的最终设置: 1. 通过 “Simulation” 确保处于最佳状态 2. “3D Garment” → “Optimize for Export” 3. 执行多边形数量优化 4. 确认纹理压缩设置

    🔧 故障排除¶

    🆕 2025 年新功能相关问题¶

    “Re-target Draping 失败”

    原因与处理方法:

    1. 头像兼容性问题:
    2. 原头像和目标头像的骨骼结构差异过大
    3. 处理: 用体型相近的头像作为中转

    4. 替代方案: 通过分阶段 Re-targeting 缩小差异

    5. 内存不足错误:

    6. 在处理大型纹理（4K 以上）时出现
    7. 处理: 预先将纹理压缩到 2K 以下

    8. 推荐: 确保有 16GB 以上 RAM

    9. UV 映射保留失败:

    10. 出现在具有复杂 UV 展开的服装上
    11. 处理: 预先执行 UV Simplification
    12. 设置: 将 Re-target 质量降到 “Medium”
    “Enhanced Auto Fitting 结果异常”

    处理方法:

    1. AI 辅助适配过拟合:
    2. 当 AI 让服装过度贴紧体型时
    3. 解决: 将 “AI-Assisted Fitting” 改为 “Conservative”

    4. 微调: 将 “Fit Tolerance” 增加 5 - 10mm

    5. 按部位优化发生冲突:

    6. 当胸、腰、臀设置彼此矛盾时
    7. 解决: 仅对有问题的部位使用 “Manual Override” 单独调整
    8. 预防: 用 “Body Zone Priority” 指定主要部位
    “Automatic Fitting Suit Creation 无法运行”

    原因检查清单:

    • 是否使用 Marvelous Designer 2025.0 及以上
    • 头像是否具有 Humanoid 绑定骨架
    • 多边形数量是否在 20K - 100K 范围内
    • 骨骼名称是否符合标准规范（Standard / Mixamo 等）

    解决步骤: 1. 在其他 3D 软件中将骨骼名称标准化 2. 将多边形数量调整到合适范围 3. 切换到 “Legacy Mode” 使用传统方式

    EveryWear 相关问题¶

    “EveryWear 处理失败”

    原因与处理方法:

    1. 系统资源不足:
    2. 确保有 16GB 以上 RAM
    3. 关闭其他应用程序

    4. 将处理质量降到 “Medium”

    5. 头像兼容性问题:

    6. 检查头像的骨骼结构
    7. 重新导出 FBX 文件

    8. 先用简单服装进行测试

    9. 网络问题:

    10. 检查网络连接
    11. 检查防火墙设置
    12. 如果处于代理环境，请调整相关设置
    “Auto Fit 结果不理想”

    处理方法:

    1. 参数调整:
    2. 调整 Fit Tolerance（范围 5 - 20mm）
    3. 提高 Quality Level

    4. 分别优化按部位设置

    5. 分阶段处理:

    6. 按“整体 → 局部”的顺序处理
    7. 只手动调整有问题的部位
    8. 通过多次 Iteration 改善结果
    “特殊体型下自动化不起作用”

    处理方法:

    1. 混合式方法:
    2. 仅对标准部分使用自动处理
    3. 对特殊部分手动处理

    4. 分阶段调整处理方式

    5. 利用参考模型:

    6. 参考体型相近的现有头像
    7. 将调整值做成模板
    8. 按部件分别处理

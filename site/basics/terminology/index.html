    🎯 Marvelous Designer 术语¶

    基础概念・操作 ★★★¶

    • 版型: 服装纸样、2D 模板 (→Garment)
    • Garment: 整体 3D 服装、成品 (→版型、模拟)
    • 模拟: 通过物理运算计算布料动作 ★★★
    • 适配: 让服装适合头像的作业
    • 缝合: 连接并缝接各个版型
    • 垂坠塑形: 立体摆放布料并创建形状
    • 放码: 尺码展开、版型调整
    • 标记点: 版型上的标记与基准点

    AI 功能相关 ★★¶

    • AI 纹理生成器: 根据关键词自动生成纹理
    • 提示词: 用于 AI 生成的指令文本（仅支持英文）
    • 实时预览: 立即查看 3D 效果的功能
    • 无缝纹理: 没有接缝的连续纹理
    • 参数调整: AI 生成时的详细设置

    动画・动作 ★★¶

    • 关键帧: 动画的基准点
    • 时间轴: 按时间轴管理动画
    • 动作捕捉: 把真实动作转成数据
    • Pose Editor: 设置静态姿势
    • 行走循环: 行走动作的循环

    渲染・输出 ★★¶

    • 渲染: 生成最终图像或视频
    • 灯光: 光源设置与照明计算
    • 材质编辑器: 详细设置材质质感
    • 纹理映射: 将图像贴到 3D 表面
    • UV 展开: 为纹理应用做 3D→2D 转换准备
    • 法线贴图: 表面的凹凸信息
    • HDRI 灯光: 360 度环境光源

    导出・文件输出 ★★★¶

    • 导出 FBX: 为 Unity 导出 3D 文件
    • 导出 OBJ: 导出通用 3D 网格
    • GLTF 导出: 用于 Web 的轻量 3D 格式
    • PZIP 文件: MD 专用项目格式
    • DXF 导出: 面向 CAD 的 2D 版型输出

    🎮 VRChat・SDK 术语¶

    基础系统 ★★★¶

    • SDK: 软件开发工具包 (→VRChat SDK3)
    • 头像: VR 空间中的 3D 角色表现
    • 世界: VRChat 中的 3D 空间与环境
    • 实例: 世界的独立运行环境
    • 房间: 多名用户聚集的共享空间

    头像结构 ★★★¶

    • 骨骼: 骨架结构、动画基准 (→绑定骨架、Humanoid)
    • 绑定骨架: 整体动画控制系统
    • Humanoid: Unity 标准人形绑定格式 ★★★
    • Armature: Blender 术语中的骨架结构
    • 网格: 3D 模型的面结构 (→多边形、三角面)

    物理・动作系统 ★★★¶

    • PhysBone: VRChat 独有的物理骨骼 ★★★
    • Dynamic Bone: 旧式物理运算系统（计划废止）
    • 碰撞: 碰撞判定与物理边界
    • 约束: 骨骼之间的约束与联动
    • IK: Inverse Kinematics，逆向运动学
    • FK: Forward Kinematics，正向运动学

    优化・性能 ★★★¶

    • 性能等级: Excellent / Good / Medium / Poor / Very Poor ★★★
    • Quest 兼容性: 面向 Meta Quest 的轻量化适配 ★★
    • 多边形数量: 3D 模型的面数、重要指标 ★★★
    • 三角面数量: 三角形面的总数
    • 骨骼数量: 骨架数量，会影响性能
    • 材质槽数量: 材质设置的数量
    • 纹理内存: 图像数据占用容量 ★★★
    • Draw Call: 绘制调用次数、优化指标

    上传・公开 ★★¶

    • 公开头像: 公开给他人使用的头像
    • 私有头像: 个人专用头像
    • 克隆头像: 复制使用他人头像
    • Fallback 头像: Quest 不兼容时的替代头像
    • 缩略图: 头像选择界面的图片

    高级功能 ★¶

    • 头像性能: 系统负载评估
    • 安全系统: 屏蔽问题内容的系统
    • 头像 3.0: 最新头像系统
    • Expression Menu: 头像表情与动作菜单
    • Parameters: 控制头像状态的变量

    🔧 Unity 术语¶

    基础要素 ★★★¶

    • GameObject: 3D 空间中的元素、实体
    • 组件: 对象的功能与行为
    • Transform: 位移、旋转、缩放信息 ★★★
    • Hierarchy: 对象层级结构显示
    • Inspector: 查看选中对象详细设置的界面
    • Scene: 工作空间、3D 环境编辑界面 ★★★
    • Game: 运行时显示界面

    Asset 管理 ★★★¶

    • Project: 管理整个项目的文件
    • Asset: 项目中的资源与素材
    • Prefab: 可复用对象模板 ★★★
    • Package: 功能扩展包
    • 脚本: C# 程序代码

    渲染・绘制 ★★★¶

    • 材质: 表面质感与外观设置 ★★★
    • Shader: 决定绘制方式与视觉效果的程序 ★★★
    • Renderer: 执行绘制的组件
    • Mesh Renderer: 用于绘制网格的 Renderer
    • Skinned Mesh Renderer: 与骨骼联动的网格绘制组件 ★★★
    • 纹理: 图像纹理、表面图像 ★★★

    动画 ★★¶

    • Animator: 动画控制系统
    • Animation Clip: 单独的动画数据
    • Controller: 动画状态管理
    • 状态机: 状态切换系统
    • Blend Tree: 动画混合系统

    灯光・光源 ★★¶

    • Light: 光源对象
    • Directional Light: 平行光（太阳光）
    • Point Light: 点光源（灯泡）
    • Spot Light: 聚光灯（手电筒）
    • 全局光照: 全局照明、间接光
    • Lightmap: 预先计算好的光照信息
    • Reflection Probe: 采集反射环境信息

    构建・输出 ★★¶

    • Build Settings: 构建设置与输出形式
    • Platform: 目标平台
    • Target: 构建目标、输出目标
    • Player Settings: Player 设置、应用信息

    📐 3D 建模基础术语¶

    基础结构 ★★★¶

    • 网格: 3D 对象的面结构
    • 多边形: 多边形面，通常为四边形
    • 三角面: 三角形面，是 GPU 处理单位 ★★★
    • 顶点（Vertex）: 网格的角与点
    • 边: 网格的边与线
    • 面: 网格的面

    建模方法 ★★¶

    • 盒式建模: 从立方体开始塑造形状的方法
    • 挤出: 把面向外推出以形成体积
    • 内插: 在面内创建新的面
    • 倒角: 把边角处理成圆滑或斜面
    • 环切: 添加环绕一圈的边
    • 合并: 合并顶点与面

    变形・修正 ★★¶

    • 雕刻: 像黏土一样塑造形体的方法
    • 重拓扑: 重建并优化网格
    • 抽面: 减少多边形数量
    • 平滑: 让表面更光滑
    • 细分: 通过分割提升细节

    UV・纹理 ★★★¶

    • UV 映射: 通过 3D→2D 展开配置纹理 ★★★
    • UV 坐标: 纹理中的位置坐标（0-1）
    • 接缝: UV 展开时的切口与边界线
    • UV 岛: UV 展开后分离的面组
    • 打包排布: 在 UV 空间中的高效排布
    • 纹素密度: 单位面积对应的纹理分辨率

    🧵 布料・纺织术语¶

    布料种类 ★★¶

    • 棉: 棉，天然材料
    • 亚麻: 麻，透气性好
    • 丝绸: 丝，高级材料
    • 羊毛: 毛，保温性高
    • 涤纶: 合成纤维，耐久性高
    • 牛仔布: 厚棉布，牛仔裤面料
    • 皮革: 皮质，较硬材料
    • 针织: 编织面料，有伸缩性

    布料特性・物理参数 ★★★¶

    • Particle Distance: 模拟精度、网格细密程度 ★★★
    • Bend: 折弯阻力、布料硬度 ★★★
    • Stretch: 伸缩阻力、布料延展性 ★★★
    • Shear: 剪切阻力、布料扭曲程度 ★★★
    • 重力: 作用于布料的向下力
    • 空气阻力: 动作时由空气产生的阻力
    • 摩擦系数: 表面滑动的难易程度

    缝制・版型 ★★¶

    • 缝份: 版型边缘的留白部分
    • 缺口记号: 用于对齐版型的标记
    • 省道: 用于塑造立体形状的折缝
    • 褶裥: 装饰性的折痕
    • 抽褶: 通过收拢布料形成的立体感
    • 斜裁: 相对布纹 45 度的方向
    • 裁剪: 按照版型裁布

    纹理・质感 ★★¶

    • 织纹: 布料编织与织造的图案
    • 起毛: 表面起绒效果
    • 光泽: 表面的反射特性
    • 透明度: 光的透过程度
    • 粗糙度: 表面的颗粒感程度
    • 法线: 表面的方向与凹凸信息

    ⚡ 性能・优化术语¶

    VRChat 性能指标 ★★★¶

    • Excellent: 最高等级、推荐水平
    • Good: 良好、一般品质
    • Medium: 普通，有轻微问题
    • Poor: 较重，需要改进 ★★
    • Very Poor: 最重，需要紧急改进 ★★★

    多边形优化 ★★★¶

    • 多边形数量: 面的总数，直接影响性能 ★★★
    • LOD: Level of Detail，按距离切换细节程度 ★★
    • 遮挡剔除: 停止绘制看不见的面
    • 视锥剔除: 排除视野外对象
    • 网格优化: 删除多余顶点与面
    • 简化: 减少多边形数量的处理

    纹理优化 ★★★¶

    • 纹理分辨率: 图像尺寸，推荐 512x512 ★★
    • 压缩格式: DXT、ASTC 等高效格式
    • Mip Map: 按距离自动调整分辨率
    • 纹理图集: 合并多个图像
    • UV 优化: 高效利用纹理空间

    PhysBone 优化 ★★★¶

    • PhysBone 组件数量: 推荐控制在 32 个以下 ★★★
    • PhysBone 统合: 集中管理多个骨骼
    • 碰撞优化: 提升碰撞判定效率
    • 更新频率: 物理计算的执行间隔
    • 影响范围: 限定 PhysBone 的生效范围

    Quest 优化 ★★¶

    • Quest 兼容: 面向 Meta Quest 的优化适配 ★★
    • 移动端渲染: 轻量绘制方法
    • 简易 Shader: 使用轻量 Shader
    • 低分辨率纹理: 面向 Quest 的画质调整

    系统负载 ★★¶

    • CPU 负载: 中央处理器的处理量
    • GPU 负载: 图形处理器的负载
    • 内存使用量: RAM 的消耗量
    • VRAM 使用量: 显存消耗量
    • Draw Call 数量: 绘制命令次数
    • 批处理: 合并绘制命令的处理

    🗂️ 文件格式・数据术语¶

    3D 文件格式 ★★★¶

    • FBX: Autodesk 格式，Unity 推荐 ★★★
    • OBJ: 通用网格格式，较轻量
    • GLTF: 面向 Web 的 3D 格式，下一代标准
    • DAE: Collada 格式，通用性高
    • 3DS: 3ds Max 格式，较老的标准
    • PLY: 多边形格式，用于研究场景

    Marvelous Designer 专用 ★★★¶

    • PZIP: MD 项目文件 ★★★
    • ZPRJ 文件: 项目设置
    • DCC 文件: 自定义控制
    • MDL 文件: 模型数据

    纹理・图像格式 ★★★¶

    • PNG: 无损压缩，支持透明 ★★★
    • JPEG/JPG: 有损压缩，适合照片
    • TGA: Targa，支持 Alpha 通道
    • EXR: HDR，高精度图像
    • DDS: DirectDraw，面向游戏优化
    • PSD: Photoshop 格式，保留图层

    Unity 专用格式 ★★¶

    • Unity Package: 功能与 Asset 的分发格式
    • Prefab: 预制体文件
    • Material: 材质设置
    • Animation: 动画剪辑
    • Scene: 场景文件

    VRChat 用格式 ★★¶

    • VRCAvatarDescriptor: 头像设置
    • VRCWorld: 世界设置
    • VRCA: 用于上传的头像文件

    🎨 渲染・Shader 术语¶

    基础渲染 ★★★¶

    • 渲染管线: 绘制处理流程
    • 前向渲染: Forward Rendering，Unity 标准方案
    • 延迟渲染: Deferred Rendering，高质量方案
    • 光栅化: 向量到像素的转换

    Shader 种类 ★★★¶

    • Standard Shader: Unity 标准 Shader ★★★
    • Unlit Shader: 无光照 Shader，较轻量
    • Transparent Shader: 支持透明的 Shader
    • Cutout Shader: 镂空透明 Shader
    • Toon Shader: 卡通着色 Shader

    VRChat Shader ★★¶

    • VRChat/Mobile: 支持 Quest 的轻量 Shader ★★
    • VRChat/Standard: 面向 VRChat 优化的标准 Shader
    • Poiyomi Shader: 高功能自定义 Shader
    • lilToon: 动画风高质量 Shader
    • UnityChan Toon Shader: 面向 UnityChan 的 Shader

    Shader 功能 ★★¶

    • Albedo: 底色、基础颜色
    • Metallic: 金属感与反射特性
    • Smoothness: 表面平滑程度
    • Normal Map: 法线与凹凸信息
    • Occlusion: 环境遮蔽与阴影信息
    • Emission: 自发光部分

    高级渲染 ★¶

    • PBR: Physically Based Rendering，基于物理的渲染
    • HDR: High Dynamic Range，高动态范围
    • 色调映射: 将 HDR 转成屏幕显示
    • 伽马校正: 对亮度进行非线性校正
    • 色彩空间: sRGB、Linear 等色彩表达方式

    Marvelous Designer 基础界面指南¶

    指南信息

    预计用时: 约60分钟 | 难度: 适合初学者 | 重要度: 必须掌握基础操作

    本页目的: 彻底理解 Marvelous Designer 的界面，从而能够高效地开展作业。

    准备事项

    • 已完成 Marvelous Designer 首次启动指南
    • Marvelous Designer 能正常运行
    • 可以创建新项目

    🎯 本次要达成的目标¶

    1. ✅ 理解主界面的全部区域
    2. ✅ 记住基础工具的位置与作用
    3. ✅ 掌握高效的操作方法
    4. ✅ 理解自定义方式
    5. ✅ 实际练习基础操作

    📐 主界面的构成¶

    Marvelous Designer 的主界面分为 5 个主要区域：

    🎨 1. 2D 版型窗口（左侧）¶

    Pattern Window 的作用

    在这里进行的作业:

    • 创建与编辑服装纸样（版型）
    • 平面作业（绘线、编辑形状）
    • 设置缝线
    • 调整纸样布局

    显示内容: - ✅ 网格: 作业参考线 - ✅ 版型片: 服装的各个部件 - ✅ 缝线: 红线（Internal Line） - ✅ 外轮廓线: 蓝线（Outline）

    基本操作: - 缩放: 鼠标滚轮 - 平移（移动）: 中键拖拽 - 显示全体: Ctrl + A → F

    🎭 2. 3D 视口（右上）¶

    3D Garment Window 的作用

    这里可以确认的内容:

    • 头像（人形模型）
    • 3D 化后的服装
    • 布料物理模拟结果
    • 实际穿着状态

    视角操作: - 旋转: 右键拖拽 - 平移（移动）: 中键拖拽 - 缩放: 鼠标滚轮 - 聚焦显示: F 键

    显示模式切换: - 1 键: 线框 - 2 键: 着色 - 3 键: 纹理显示 - 4 键: 渲染显示

    !!! tip "视角操作技巧"
    按住右键移动鼠标后，可以围绕头像进行旋转。刚开始可能需要稍微练习一下。

    🎛️ 3. 3D Object Browser（右下）¶

    Object Browser 的用法

    显示内容:

    • Avatar: 已导入的头像
    • Garments: 已创建的服装列表
    • Trims: 纽扣、拉链等装饰

    基本操作: - 选择: 点击对象名称 - 显示/隐藏: 点击 👁️ 图标 - 锁定: 用 🔒 图标锁定编辑

    重要功能: - 管理对象层级 - 控制显示与隐藏 - 管理选中状态

    🛠️ 4. 工具栏（顶部）¶

    主要工具的位置

    左侧: 文件操作 - New（新建）: 创建新项目 - Open（打开）: 打开现有文件 - Save（保存）: 保存项目

    中间: 2D 绘图工具 - Arrow（选择）: 选择与移动对象 - Rectangle（矩形）: 创建矩形版型 - Circle（圆）: 创建圆形版型 - Polygon（多边形）: 创建复杂形状 - Curve（曲线）: 曲线工具

    右侧: 模拟控制 - ▶️ Simulate: 开始/停止模拟 - ⏸️ Pause: 暂停模拟 - 🔄 Reset: 重置到初始状态

    工具的选择方式

    • 用鼠标点击
    • 使用键盘快捷键（鼠标悬停在各工具上时会显示）
    • 从右键菜单访问

    ⚙️ 5. 属性面板（右侧或下方）¶

    Property Editor 的使用

    显示的设置内容（会根据当前选中的对象变化）:

    选中版型时: - 物理特性: 布料的硬度、重量等 - 模拟设置: 质量、精度 - 显示设置: 颜色、透明度

    选中头像时: - 姿势设置: 调整姿态 - 尺寸调整: 调整头像比例 - 碰撞设置: 与布料的碰撞判定

    重要设置项: - Fabric Preset: 布料种类（Cotton、Silk、Denim 等） - Particle Distance: 模拟精度 - Internal Lines: 缝线设置

    属性面板的位置

    根据版本或布局设置不同，它可能显示在右侧或下方。如果找不到，请从 Window 菜单选择 Property Editor。

    📋 菜单栏详细说明¶

    File（文件）菜单¶

    常用的 File 菜单项

    基本操作: - New Project: 创建新项目 - Open: 打开现有项目（.zprj 格式） - Save: 保存当前作业 - Save As: 另存为

    导入/导出: - Import: 导入外部文件（FBX、OBJ 等） - Export: 导出为其他格式（FBX、OBJ、Alembic 等）

    最近使用的文件: - Recent Files: 最近打开的项目列表

    文件管理技巧

    在每个作业节点都养成使用 Ctrl + S 保存的习惯。

    Edit（编辑）菜单¶

    编辑功能的使用

    基本编辑: - Undo (Ctrl+Z): 撤销操作 - Redo (Ctrl+Y): 重做 - Cut/Copy/Paste: 剪切、复制、粘贴

    环境设置: - Preferences: 全局设置（语言、单位、自动保存等）

    选择功能: - Select All: 全选 - Deselect All: 取消选择

    2D Pattern（2D 版型）菜单¶

    专用于版型制作的功能

    形状创建: - Add Rectangle: 添加矩形 - Add Circle: 添加圆形 - Add Polygon: 添加多边形

    版型编辑: - Edit Pattern: 编辑版型形状 - Transform: 变形（移动、旋转、缩放）

    缝合设置: - Sewing: 设置并执行缝线 - Free Sew: 自动缝合

    版型制作的基本流程

    创建形状 → 编辑与调整 → 设置缝线 → 执行模拟

    3D Garment（3D 服装）菜单¶

    3D 显示与控制

    模拟控制: - Simulate: 开始/停止模拟 - Reset 3D Arrangement: 重置 3D 排列 - Reset 2D Arrangement: 重置 2D 排列

    显示设置: - Display: 切换显示模式 - Show/Hide: 切换对象显示

    物理设置: - Simulation Properties: 模拟详细设置 - Physical Properties: 物理特性设置

    Avatar（头像）菜单¶

    头像管理功能

    头像操作: - Load Avatar: 导入头像文件 - Edit Avatar: 调整头像姿势与尺寸 - Show/Hide Avatar: 切换头像显示

    姿势控制: - Pose: 预设姿势 - Edit Pose: 创建自定义姿势

    关于头像导入

    FBX 文件通过 Avatar → Load Avatar 导入。后面会详细说明。

    ⌨️ 重要键盘快捷键¶

    应当记住的基础快捷键

    基本操作: - Ctrl + N: 新建项目 - Ctrl + O: 打开文件 - Ctrl + S: 保存 - Ctrl + Z: 撤销 - Ctrl + Y: 重做

    显示控制: - F: 聚焦显示到选中对象 - Ctrl + A: 全选 - 1, 2, 3, 4: 切换显示模式

    模拟: - Space: 开始/停止模拟 - Ctrl + R: 重置模拟

    工具选择: - A: Arrow（选择工具） - R: Rectangle（矩形工具） - C: Circle（圆形工具）

    快捷键的优势

    记住快捷键后，作业效率会大幅提升。先从最基础的几个开始记就够了。

    🔄 与 Blender 和 Unity 统一操作设置¶

    与 Blender 统一操作¶

    设置成更接近 Blender 的操作手感

    Blender 的 Industry Compatible 键位设置:

    如果想在 Blender 中获得更接近 Marvelous Designer 的操作手感： 1. Edit → Preferences → Keymap 2. 选择 Industry Compatible 3. 这样可以提升与 Maya、Unreal Engine 等其他 3DCG 软件的兼容性

    可统一的操作: - Alt + 左键拖拽: 3D 视图旋转（两款软件通用） - Alt + 中键拖拽: 平移（两款软件通用） - Alt + 右键拖拽: 缩放（两款软件通用） - Space: 模拟控制（MD）/ 播放控制（Blender） - F: 聚焦显示（两款软件通用）

    高效工作流

    从 Marvelous Designer 切换到 Blender 时，建议在导出 OBJ 时使用“单一对象”“不合并顶点”“薄厚度”“启用统一 UV 坐标”等设置，并在 Blender 中以默认立方体（2m³）为基准调整比例。

    与 Unity 的联动设置¶

    面向 Unity 的优化设置

    通用操作要素: - 中键拖拽: 移动 3D 视图（两款软件通用） - Alt + 左键: 轨道旋转操作 - 缩放操作: 鼠标滚轮（两款软件通用） - F: 聚焦显示（Object focus）

    Marvelous Designer → Unity 工作流: 1. 以 FBX 格式导出: 考虑 Unity 兼容性 2. 统一比例: 对齐 Unity 标准（1unit = 1m） 3. 保留 UV 坐标: 确保纹理映射一致 4. 保留骨骼结构: 兼容动画

    推荐设置: - 导出时统一单位（以米为基准） - 确保材质设置兼容 - 正确转换物理属性

    使用专用键盘¶

    用于提效的外部设备设置

    推荐键盘设置: 部分高级用户会使用 Razer Tartarus V2 等键盘设备来提升 Marvelous Designer 特有操作的效率。

    设置示例: - 把常用工具（Rectangle、Circle、Arrow）分配到专用按键 - 把模拟控制放在拇指容易按到的位置 - 通过切换图层快速在 2D / 3D 操作间切换

    自定义的优势

    如果你同时使用多款 3DCG 软件，保持统一的操作手感可以大幅提升作业效率与学习效果。

    🎨 界面自定义¶

    调整布局¶

    把画面调整到更适合作业的状态

    调整面板大小: 1. 将鼠标移到面板之间的分隔线 2. 光标变化后拖拽以调整大小 3. 根据作业内容调整到合适的尺寸

    推荐布局: - 2D 版型窗口: 占屏幕的 40-50% - 3D 视口: 占屏幕的 40-50% - Object Browser: 保持适中大小 - Property Editor: 需要时显示

    保存布局: 1. Window → Save Layout 2. 输入名称保存（例如：服装制作） 3. 可通过 Window → Load Layout 调出

    不同屏幕尺寸的推荐设置

    • 小屏幕: 让 3D 视口稍大一些
    • 大屏幕: 2D 和 3D 采用相同比例
    • 双显示器: 在副屏显示 3D 视图

    自定义工具栏¶

    更高效的工具排列

    常用工具的摆放: 1. Window → Customize Toolbar 2. 将常用功能拖放到工具栏中 3. 删除不常用的功能，让布局更简洁

    推荐自定义方式: - 把模拟控制按钮放在更容易看到的位置 - 将常用的版型制作工具集中摆放 - 把文件操作集中在左侧

    🔧 实践练习：界面操作¶

    练习1：确认基本操作¶

    画面操作练习

    3D 视口练习: 1. 用右键拖拽从不同角度查看头像 2. 用鼠标滚轮缩放 3. 按 F 键显示全体 4. 用 1、2、3、4 键切换显示模式

    2D 版型窗口练习: 1. 用中键拖拽移动画面 2. 用鼠标滚轮缩放 3. 点击空白区域，使其处于未选择任何内容的状态

    已经熟悉操作

    这些基本操作在后续作业中会频繁使用。请练习到能够顺畅完成为止。

    练习2：创建简单版型¶

    使用工具进行作业练习

    创建矩形版型: 1. 选择 Rectangle 工具（工具栏或按 R 键） 2. 在 2D 版型窗口中点击并拖拽 3. 创建一个合适大小的矩形 4. 切回 Arrow 工具（按 A 键）

    创建圆形版型: 1. 选择 Circle 工具（按 C 键） 2. 在 2D 版型窗口中点击并拖拽 3. 创建圆形 4. 切回 Arrow 工具

    执行模拟: 1. 按 Space 键 2. 刚创建的版型会显示在 3D 视口中 3. 确认它在重力作用下下落的状态 4. 再次按 Space 键停止模拟

    基础操作完成

    至此你已经理解了 Marvelous Designer 基本操作的整体流程。

    练习3：编辑属性¶

    修改设置的练习

    修改布料特性: 1. 选中已创建的矩形 2. 在 Property Editor 中查看 Physical Property 3. 尝试修改 Fabric Preset（例如：Cotton → Silk） 4. 执行模拟，确认差异

    更改颜色: 1. 选中版型 2. 在 Property Editor 的 Color 中修改颜色 3. 在 3D 视口中确认颜色变化

    设置的效果

    选择不同的布料类型后，硬度和重量都会变化，模拟结果也会随之改变。

    🔧 故障排查¶

    界面相关¶

    “面板不见了”

    处理方法: 1. 从 Window 菜单重新显示对应面板 2. 通过 Window → Reset Layout 恢复初始布局 3. 拖拽面板边界调整大小

    “工具栏没有显示”

    处理方法: 1. 在 View → Toolbars 中启用对应工具栏 2. 通过 Window → Customize Toolbar 重新设置

    “操作很卡、很慢”

    处理方法: 1. 在 Edit → Preferences → 3D Graphics 中降低图形设置 2. 降低显示质量（按 2 键切换到着色显示） 3. 隐藏不需要的对象

    操作相关¶

    “模拟不运行”

    处理方法: 1. 确认是否已创建版型 2. 在 3D Garment → Simulation Properties 中确认 Gravity（重力）是否启用 3. 确认版型尺寸是否合适

    显示模式一览¶

    键位 显示模式 用途 Blender 对应
    1 Wireframe 仅显示线框（更轻量） Z → 1（Wireframe）
    2 Shaded 带明暗的显示（标准） Z → 2（Solid）
    3 Textured 纹理显示 Z → 3（Material Preview）
    4 Rendered 高质量渲染 Z → 4（Rendered）

    与其他 3DCG 软件的兼容性¶

    统一数据格式: - FBX: 与 Unity、Maya、3ds Max 兼容性高 - OBJ: 兼容 Blender、ZBrush 等广泛软件 - Alembic: 用于高精度动画与模拟

    推荐工作流: 1. 在 Marvelous Designer 中制作服装并执行模拟 2. 根据用途导出合适格式（按需选择 FBX / OBJ） 3. 在 Blender 或 Unity 中进行最终调整与集成 4. 根据需要，继续面向 VRChat 或其他平台进行优化

    泳装制作指南（中级～高级）¶

    指南信息

    预计用时：约 4-6 小时 | 难度：中级～高级 | 重要度：高级技术学习的集大成

    本页目标：整合到目前为止掌握的基础技术，通过制作最具技术挑战性的泳装，掌握贴身服装制作的高级技术。

    本指南能掌握的技能

    • ✅ 与体型严丝合缝的版型设计技术
    • ✅ 弹性面料的高级物理设置
    • ✅ 顺着复杂身体曲线进行立体构建
    • ✅ 面向 VRChat 的高性能贴身服装优化
    • ✅ 高级故障排查方法
    • ✅ 专业级收尾技术

    前置准备的重要性

    泳装制作需要非常高级的技术。强烈建议在开始前完成以下前置准备：

    必备的前置学习

    • 完成T 恤制作指南并理解基础版型
    • 完成裙子制作指南并掌握曲线版型
    • 仔细阅读布料物理特性设置，理解弹性面料
    • 头像已正常导入 Marvelous Designer
    • 预留 4-6 小时的连续专注时间

    🎯 要制作的泳装规格¶

    本次项目目标¶

    完成目标

    要制作的泳装类型：连体竞泳泳装风格 

    设计规格:
├── 正面: 圆领，覆盖范围适中
├── 背面: 露背，带肩带
├── 材质感: 具有光泽的弹性面料
├── 贴合: 紧贴身体，但仍易于活动
└── 用途: VRChat 水边活动、拍照

    技术目标： - 完美贴合头像体型 - 重现自然的伸缩与恢复力 - 保证在 VRChat 中舒适动作 - 实现高质量外观与真实感

    为什么泳装最难？¶

    泳装制作的特殊挑战

    1. 极高的贴合要求 - 精确跟随身体所有曲线 - 不允许明显缝隙或悬空 - 动作时也要保持形状

    2. 特殊材质特性 - 四向弹性（纵横都能拉伸） - 形状记忆与恢复力 - 光泽与通透感的表现

    3. 复杂立体结构 - 精确再现 3D 曲面 - 整合胸线、腰线、臀线 - 自然处理开口部（颈部、手臂、腿部）

    4. VRChat 特有课题 - 紧身服装的碰撞判定问题 - 避免物理演算负载过高 - 在穿着状态下保持稳定

    🚀 Phase 1: 高级设计与规划（45-60 分钟）¶

    测量与分析体型¶

    步骤 1-1：精确测量体型

    获取头像的重要尺寸：

    由于 VRChat 头像差异很大，精确测量是必要前提。

    测量部位与常见数值： 

    🔸 胸围: 85-95cm（最宽处）
🔸 下胸围: 75-85cm（胸部下方）
🔸 腰围: 60-70cm（最细处）
🔸 臀围: 90-100cm（最宽处）
🔸 肩宽: 40-45cm
🔸 衣长参考: 65-75cm（从肩到裆下）

    在 Marvelous Designer 中的测量方法： 1. 选择 Measure 工具 2. 在各个测量部位准确测量 3. 将结果记录到记事本（后续参考） 4. 从多个角度观察头像，掌握其体型特征

    提高测量精度的技巧

    • 旋转头像，从多个角度确认
    • 检查是否对称（左右差异）
    • 特别关注曲线变化剧烈的区域
    • 仔细观察泳装会暴露出的部位轮廓

    规划贴合策略¶

    步骤 1-2：设计贴合区域

    分析泳装的关键贴合区域：

    Zone A: 胸部区域（最重要） - 最重视保持形状 - 需要支撑结构 - 动作时必须稳定

    Zone B: 腰线（设计核心） - 决定体型观感的重要部分 - 需要在适度收紧与舒适之间平衡 - 要适应坐下、弯腰等动作

    Zone C: 臀线（复杂曲线） - 精确贴合立体曲面 - 自然处理腿部开口 - 确保坐姿舒适性

    Zone D: 开口部（技术难点） - 颈部、手臂、腿部周围 - 既不能太紧，也不能太松 - 边缘要收得优美

    确定材质策略¶

    步骤 1-3：选择泳装材质

    推荐物理特性设置： 

    基础设置:
├── Fabric Type: Lycra/Spandex
├── Stretch-Weft: 40-50%（横向拉伸）
├── Stretch-Warp: 35-45%（纵向拉伸）
├── Thickness: 0.5-0.8mm
├── Weight: 150-200g/m²
└── Recovery: 95% 以上

    颜色与质感方向： - 纯色：深蓝、黑、红等经典色 - 光泽感：泳装常见的缎面反射 - 图案：考虑后续扩展，本次先保持简单

    🛠️ Phase 2: 高级版型制作（90-120 分钟）¶

    构建基础版型¶

    步骤 2-1：创建前身片立体版型

    构建前身片的复杂形状：

    泳装前身片不能用简单长方形完成，必须做出顺着身体曲线的复杂形状。

    方法 1：基于测量值设计： 1. 用 Polygon 工具 勾出基础轮廓 2. 制作反映关键尺寸的草图形状： - 上部：肩宽的一半 + 余量（22-25cm） - 胸部：胸围的一半（42-47cm） - 腰部：腰围的一半（30-35cm） - 长度：按测量值（65-75cm） 3. 用 Edit Pattern 工具调整为曲线

    关键形状要点： - Neckline：贴合颈部但不过紧的曲线 - Armhole：顺着手臂根部的立体曲线 - 胸线：顺着胸型自然隆起 - 腰线：向身体最细处收束 - 腿部开口：顺着腿根形成自然曲线

    制作立体版型的技巧

    不要一开始就追求完美。先做出基础形状，再逐步调整。请频繁确认贴合效果。

    步骤 2-2：后身片的特殊结构

    后身片设计时的注意点：

    泳装后身片与前身片有很大差异：

    特殊要素： - 大面积露背：创造优美轮廓 - 肩带结构：稳定连接肩部 - 低背线：增强背面观感

    制作步骤： 1. 以前身片为基础复制出后身片 2. 大胆裁掉背部区域（做出露背）： - 上部：开到肩胛骨下方（约 15-20cm 的开口） - 下部：保持到腰线 3. 添加肩带部分： - 宽度：2-3cm - 长度：从肩部到露背处（约 25-30cm） - 左右对称制作两条

    调整后身片曲线： - 顺着脊柱形成自然线条 - 注意肩胛骨形状 - 让腋下到背部的过渡自然

    高级立体构建技术¶

    步骤 2-3：Dart 与立体化技法

    构建胸部区域立体感：

    要用平面版型做出立体泳装，Dart（省道） 技法必不可少。

    设置胸 Dart： 1. 选中前身片版型 2. 用 Edit Pattern 工具进入编辑模式 3. 以胸点（最突出的部分）为基准： - 侧 Dart：从侧边指向中部的三角形（约 3-5cm） - 下 Dart：从下往上指向胸点的三角形（约 2-4cm） 4. 通过闭合 Dart 实现立体化

    Dart 布置理论： 

    胸 Dart 的布置原则:
├── 侧 Dart: 处理腋下多余布量
├── 下 Dart: 处理下侧多余布量
├── 角度: 向胸点收束
└── 深度: 按体型调整（通常 3-5cm）

    创建 Dart 时的注意点

    Dart 的微调非常关键。太深会不自然，太浅则贴合不足。

    步骤 2-4：边缘处理与开口部

    做出优美的开口部：

    泳装的质量，很大程度上由开口部（领口、袖口、腿口）的美观决定。

    Neckline（颈部周围）： 1. 选中前后身片的领口部分 2. 用 Curve 工具调整成顺滑曲线 3. 让其贴合颈部但不压迫（约留 1-2cm 余量） 4. 调整前后高低（前面稍高，后面稍低）

    Armhole（手臂周围）： 1. 创建连接前后身片手臂区域的曲线 2. 开口要足以保证手臂活动 3. 同时又不能太大以免不稳定

    Leg Opening（腿部周围）： 1. 这是技术上最难的部分 2. 实现从臀线到腿部的自然过渡 3. 兼顾坐姿舒适性与优美轮廓

    🧵 Phase 3: 特殊缝合与材质设置（60-90 分钟）¶

    弹性面料的缝合技术¶

    步骤 3-1：适应 Stretch 的缝合线

    泳装特有的缝合课题：

    与一般服装不同，泳装缝合必须始终考虑拉伸。

    Internal Line 的特殊设置： 1. 创建缝合线前的准备： - 将两块版型摆放在适度拉伸状态 - 预先测量待缝部分长度 - 将缝合线设置得比实际短 5-10%（为伸缩预留空间）

    主要缝合线： 

    按优先顺序缝合:
1️⃣ Side Seam（最重要）
2️⃣ Shoulder Seam（确保稳定）
3️⃣ 肩带连接（与后身片整合）
4️⃣ 边缘处理（完成开口部）

    执行缝合： 1. 用 Internal Line 工具谨慎绘制各条缝合线 2. 在 Free Sew 前务必确认长度 3. 若报错，优先调整长度 4. 一条一条缝合，并每次都确认贴合效果

    Stretch 缝合成功的关键

    不要着急，一条一条缝，并在每个阶段确认模拟结果，是成功的关键。

    高级物理特性设置¶

    步骤 3-2：泳装用物理属性

    专业级物理设置：

    泳装的物理设置比一般服装复杂得多。

    基础物理特性： 

    泳装推荐设置:
┌─ Fabric Properties ─┐
├─ Preset: Custom     │
├─ Stretch-Weft: 45%  │（横向拉伸）
├─ Stretch-Warp: 40%  │（纵向拉伸）
├─ Shear: 30°         │（斜向变形）
├─ Bend: 0.1          │（弯曲硬度：偏柔软）
├─ Density: 0.15      │（轻量设置）
└─ Thickness: 0.6mm   │（薄料的真实感）

    为 VRChat 优化而做的调整： 

    性能设置:
├─ Particle Distance: 10（质量与轻量化平衡）
├─ Self Collision: OFF（轻量化）
├─ Internal Collision: 通过 Layer 控制
└─ Simulation Quality: Medium（保持最低必要水准）

    应用设置的步骤： 1. 选中所有版型（Ctrl + A） 2. 在 Property Editor 中谨慎输入各项数值 3. 使用 Preset Save 保存设置（便于复用） 4. 细小调整放到后续模拟阶段进行

    实现多层结构（可选）¶

    步骤 3-3：添加 Lining（里布）

    用于专业品质的里布结构：

    正式泳装通常有里布（Lining），它可以带来更自然、更舒适的穿着感。

    里布的作用： - 控制通透感（防止透视） - 让贴肤触感更柔和 - 提高结构稳定性 - 提升真实感

    实现方法： 1. 复制表布： - 选中当前所有版型并复制 - 将 Layer 改为 Lining - 调整为比表布小 2-3mm

    1. 里布专用物理设置： 

      Lining 设置:
├─ 更柔软（Bend: 0.05）
├─ 更轻量（Density: 0.1）
├─ 更薄（Thickness: 0.3mm）
└─ 肤色或白色

    2. 与表布整合：

    3. 在边缘位置连接表布与里布
    4. 用 Binding 技法收出更漂亮的成品边
    5. 确认模拟时两者的相互作用

    里布属于高级技术

    Lining 是很高级的技法。建议先只用表布完成作品，熟悉后再挑战。

    🎭 Phase 4: 复杂适配与调整（60-75 分钟）¶

    紧身版型特有的模拟¶

    步骤 4-1：执行首次模拟

    泳装模拟的特殊性：

    泳装模拟与一般服装有不同的课题：

    预期的首次结果： - ✅ 理想情况：适度贴合，形状自然 - ⚠️ 常见情况：局部过紧或过松 - ❌ 需要调整：明显变形或穿透

    模拟执行步骤： 1. 最终确认版型摆放： - 所有版型都正确摆放在头像周围 - 缝合线已正确处理 - Layer 设置正确

    1. 分阶段模拟：
    2. 第一阶段：按 空格键 开始，观察约 30 秒
    3. 第二阶段：暂停并定位问题
    4. 第三阶段：按需调整后再次执行

    首次评估检查点： 

    模拟评估项目:
🔸 整体贴合: 是否顺着身体曲线
🔸 胸部区域: 是否保持自然形状
🔸 腰线: 收紧程度是否合适
🔸 开口部: 边缘是否自然美观
🔸 肩带: 是否以适当张力发挥作用
🔸 动作: 是否能跟随头像姿势变化

    高级适配调整技法¶

    步骤 4-2：局部调整技术

    紧身服装特有的调整方法：

    像泳装这种贴身服装，局部调整极其重要。

    胸部区域调整： 

    问题: 胸部太紧
处理:
├─ 将 Bust Dart 调浅（调整 1-2mm）
├─ 将 Side Seam 向外移动（2-3mm）
└─ 放松 Under Band

问题: 胸部太松
处理:
├─ 加深 Bust Dart
├─ 将 Side Seam 向内移动
└─ 收紧 Under Band

    腰线调整： 

    问题: 腰部太紧
处理:
├─ 调整整体 Scale（105-108%）
├─ 仅对腰部局部放大
└─ 微调缝合线

问题: 腰部太松
处理:
├─ 调整整体 Scale（95-98%）
├─ 仅对腰部局部缩小
└─ 考虑追加 Dart

    开口部调整： 

    Neck / Arm / Leg Opening 调整:
├─ 调整 Edge Length（约 ±2-5mm 的微调）
├─ 微调 Curve Shape（更自然的曲线）
└─ 调整 Tension（优化束缚感）

    动作测试与性能确认¶

    步骤 4-3：在复合动作中验证

    按 VRChat 预期动作进行验证：

    泳装会在多种动作下使用，因此必须从多个维度验证。

    测试动作序列： 1. 基础姿势： - 站姿下的整体平衡 - 确认左右对称 - 确认自然下垂感

    1. 手臂动作：
    2. 双臂上举（万岁姿势）
    3. 确认肩部与腋下的跟随性
    4. 双臂向前伸

    5. 确认 Armhole 是否合适

    6. 躯干动作：

    7. 前倾（鞠躬）姿势
    8. 确认腰部弹性
    9. 身体左右扭转

    10. 确认露背部分是否依然优美

    11. 坐姿测试（重要）：

    12. 模拟坐在椅子上的状态
    13. 检查臀部与腰部舒适性
    14. 确认没有不自然的拉扯

    性能监控： 

    VRChat 性能检查:
├─ 模拟负载: 稳定保持 60FPS
├─ 多边形数: 在合理范围内（<2000 polygon）
├─ 物理演算负载: Medium 以下
└─ 碰撞检测: 正常工作

    最终质量检查与微调¶

    步骤 4-4：收尾到专业品质

    🔧 高级故障排查¶

    泳装制作常见问题与解决方案¶

    “泳装会穿透头像”

    原因分析： - 紧身设置不合适 - 物理演算中的碰撞检测不足 - 版型尺寸不合适

    分阶段解决方法：

    Step 1: 检查基础设置 

    Collision 设置检查:
├─ Self Collision: OFF→ON
├─ Avatar Collision: ON，并设置合适余量
├─ Collision Thickness: 调到 3-5mm
└─ Internal Collision: 通过 Layer 控制

    Step 2: 调整版型尺寸 - 整体放大 2-3%，留出余量 - 局部调整问题区域 - 调整 Dart 深度

    Step 3: 重新检查物理设置 - 调整 Bend 到 0.1-0.3（如果过硬） - 将 Stretch 稍微放宽（50-60%） - 临时提高 Simulation Quality 进行确认

    “动作时泳装会不自然变形”

    原因分析： - 弹性设置与真实泳装材质不匹配 - 对动作的跟随性不足 - 缝合线约束过强

    解决方法：

    重新调整物理特性： 

    提升动作跟随性的设置:
├─ Stretch-Weft: 50-60%（提高横向弹性）
├─ Stretch-Warp: 45-55%（纵向也更柔软）
├─ Shear: 35-45°（增加斜向自由度）
└─ Air Damping: 0.02-0.05（让动作更顺滑）

    重新评估缝合线： - 调整 Internal Line 的张力 - 删除局部缝合线（避免过度约束） - 使用更自然的缝合顺序

    “VRChat 里性能有问题（FPS 下降）”

    性能诊断：

    定位负载来源： - 确认主要原因是否是模拟负载 - 确认多边形数量（目标：<2000） - 确认纹理尺寸（目标：1024x1024 以下）

    分阶段轻量化：

    Level 1: 基础轻量化 

    轻量化设置:
├─ Particle Distance: 15-20（降低粒子密度）
├─ Simulation Quality: Low-Medium
├─ Self Collision: OFF（减轻负载）
└─ Update Frequency: 30fps（减半）

    Level 2: 高级轻量化 - 简化复杂曲线 - 删除不必要的 Dart - 优化纹理分辨率 - 考虑实现 LOD（Level of Detail）

    “换到其他头像时贴合有问题”

    应对头像差异：

    对比分析测量值： - 详细测量原头像与新头像 - 找出体型差异特征 - 为贴合问题区域设定优先级

    高效调整方法：

    整体调整方式： - 通过 Scale 做基础适配（90-110% 范围） - 局部调整关键区域 - 重设 Dart 位置与深度

    局部调整方式： 

    按部位调整优先级:
1️⃣ 胸部区域（最高优先级）
2️⃣ 腰线（形状关键）
3️⃣ 臀线（舒适性）
4️⃣ 开口部（舒适性）

# 模块化服装系统

## 概要

借助 Marvelous Designer 2025 新引入的 **Modular Library** 功能，现在可以更高效地创建服装变体。工作流也从过去每次都从零开始重做，演进为通过组合 **Block Sets** 来制作服装的模块化方式。

!!! tip "建议学习时间"
    - **基础理解**: 30 分钟
    - **实战工作流**: 1 到 2 小时
    - **进阶技术**: 3 到 4 小时

## 模块化系统的基础概念

### 1. 模块化库（Modular Library）

这是一个可以**从已保存的 Garment 组或 Block Sets 中创建新变体**的系统。

- 将服装各部位模块化后保存到库中
- 组合现有部件，高效制作新设计
- 与 CLO-SET 集成后，Asset 管理也更方便

### 2. Block Sets

这是将构成服装的**各个部位做成模块块**的系统。

**主要块元素**:
- **身片（Body）**: 前身片、后身片
- **袖子（Sleeve）**: 长袖、短袖、无袖
- **裙装 / 裤装**: 长款、短款、喇叭型
- **领子（Collar）**: 圆领、尖领、立领

## 基础工作流

### 步骤 1: 开始模块化模式

1. 选择 **File** → **New** → **Modular Template**
2. 选择服装类别和模块模板类型
3. 选中的模块模板会显示在 2D 窗口中

### 步骤 2: 放置版型

1. 从新增的 Garment 文件中点击并拖动版型
2. 将版型放到对应的 Block Box 中
3. 当版型与 Block Box 建立链接后，方框会变成深灰色

### 步骤 3: 缝合与模拟

1. **Block Box 默认已经完成缝合**
2. 如有需要，再设置额外的缝合线
3. 运行模拟，确认服装表现

### 步骤 4: 保存块文件

1. 通过 **File** → **Save Block** 保存为块文件
2. 注册到库中，便于后续重复使用

## 面向 VRChat 的优化要点

### 模型姿势推荐设置

!!! warning "重要设置调整"
    VRC 对应模型通常以 T 姿势制作，但**强烈建议在 Marvelous Designer 中以 A 姿势进行作业**。

**A 姿势的优点**:
- 更自然的布料模拟
- 减少肩部附近的布料卡顿
- 抑制不自然褶皱的产生

### 与 EveryWear 联动

**EveryWear** 是面向游戏引擎和实时平台的导出工具集。

**主要功能**:
- 针对 VRChat SDK 的优化导出
- 自动调整多边形数量
- 优化 UV 映射
- 调整物理演算参数

## 实用应用示例

### 制作校服套装

1. **基础块**: 标准西装外套和裙子
2. **变体**: 无袖、短袖、冬装规格
3. **自定义**: 校徽、蝴蝶结、袜子长度

### 休闲服装套装

1. **上装块**: T 恤、长袖、背心
2. **下装块**: 牛仔裤、短裤、裙子
3. **配件**: 连帽外套、开衫

## 常见问题与解决方法

### Q: 块与块之间的缝合不顺利

**A:** 请检查 Block Box 之间的缝合线，并调整版型的边距设置。

### Q: 模拟结果不自然

**A:** 请确认以下几点：
- 模型是否处于 A 姿势
- 布料物理属性是否设置合适
- 布料之间的干涉设置是否合理

### Q: 导出时多边形数量过多

**A:** 请使用 EveryWear 的优化设置，将其调整到 VRChat 推荐的多边形数量范围内。

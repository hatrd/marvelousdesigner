# 模型导入指南

!!! note "指南信息"
    **预计用时**: 约 20 分钟 | **难度**: 适合初学者 | **重要度**: 必做

**本页目标**: 将你的 VRChat 模型（FBX 文件）导入 Marvelous Designer，并完成服装制作前的准备。

!!! info "开始前准备"
    - 已理解 [Marvelous Designer 基础界面](../basics/md-interface.md)
    - 已准备好 VRChat 模型的 FBX 文件
    - Marvelous Designer 可以正常运行

## 🎯 本次要达成的目标

1. ✅ 正确将 FBX 文件导入 Marvelous Designer
2. ✅ 合理设置模型的尺寸和姿势
3. ✅ 调整到适合制作服装的状态
4. ✅ 理解导入设置，方便之后顺畅重复操作

## 📁 文件准备

### 步骤 1: 确认 FBX 文件

!!! example "步骤 1-1: 提前检查文件"

    导入模型前，先确认 FBX 文件的状态。

    **检查项目**:

    - ✅ **文件大小**: 1MB 到 50MB 左右属于正常范围
    - ✅ **文件名**: 仅使用英数字，避免日文或中文
    - ✅ **保存位置**: 放在 `VRChat衣装制作/Avatars/` 文件夹中
    - ✅ **文件完整性**: 文件未损坏

    **理想的文件名示例**:
    - ✅ `MyAvatar_v1.fbx`
    - ✅ `avatar_female_base.fbx`
    - ❌ `私のアバター.fbx` （日文文件名）
    - ❌ `avatar file (new).fbx` （包含空格和括号）

!!! warning "关于 FBX 文件"
    直接使用 VRChat 里使用的模型文件即可，不需要额外转换或特殊准备。

## 🚀 模型导入步骤

### 步骤 2: 在 Marvelous Designer 中导入

!!! example "步骤 2-1: 执行导入"

    1. **启动 Marvelous Designer**，打开一个新项目
    2. 在顶部菜单点击 **“Avatar”** → **“Load Avatar”**
    3. 打开文件选择对话框
    4. 选择你的 FBX 文件后点击 **“打开”**

    **开始导入后**:
    - 会显示进度条
    - 处理通常需要 30 秒到 2 分钟
    - 请等待直到处理完成

!!! tip "导入过程中的注意事项"
    导入时不要执行其他操作，否则可能中断处理流程。

### 步骤 3: 确认导入结果

!!! example "步骤 3-1: 检查模型显示"

    导入完成后，确认 3D 视口：

    **正常导入时应看到**:
    - ✅ 模型显示在 3D 视口中
    - ✅ 以 T 姿势（双臂水平展开）显示
    - ✅ 整个模型位于画面内
    - ✅ 纹理（服装颜色、皮肤颜色）显示正常

    **显示调整方法**:
    1. 按 **“F”** 键让模型适配到画面中
    2. 按住鼠标右键拖动，从不同角度检查模型
    3. 按 **“2”** 键切换到着色显示

!!! success "导入成功的判断标准"
    如果模型显示正常，并且可以辨认脸部和身体特征，就说明导入成功。

### 步骤 4: 调整模型设置

!!! example "步骤 4-1: 检查基础设置"

    在 3D Object Browser 中检查并调整模型设置：

    **在 Object Browser 中确认**:
    1. 查看右下方的 **3D Object Browser**
    2. 在 **“Avatar”** 区块中确认已显示模型名称
    3. 点击模型名称，使其处于选中状态

    **在 Property Editor 中可调整的项目**:
    - **Scale**: 模型尺寸（通常保持 `1.0` 即可）
    - **Show Avatar**: 切换显示/隐藏
    - **Avatar Opacity**: 透明度（需要透视观察服装时可调整）

!!! example "步骤 4-2: 调整姿势（可选）"

    如有需要，也可以调整模型姿势：

    **修改姿势的步骤**:
    1. 点击 **“Avatar”** 菜单 → **“Edit Avatar”**
    2. 选择 **“Pose”** 标签页
    3. 从预设姿势中选择：
       - **T-Pose**: 基础姿势（推荐）
       - **A-Pose**: 双臂稍微下放的姿势
       - **Natural**: 自然站立姿势

    **进行服装制作时，T-Pose 最合适**

## 🔧 故障排查

### 常见问题与解决方法

??? question "“无法导入 FBX 文件”"
    **原因与处理方法**:

    1. **文件路径问题**:
       - 确认文件路径中不包含日文或中文
       - 将文件移动到更短的路径位置

    2. **文件损坏**:
       - 用其他软件（如 Blender）确认 FBX 文件能否正常打开
       - 重新下载或重新导出原始文件

    3. **版本兼容性**:
       - 如果 FBX 文件版本过旧，可能需要重新导出

??? question "“模型不显示 / 看不见”"
    **处理方法**:

    1. **检查显示设置**:
       - 在 Object Browser 中确认模型的 👁️ 图标是否已启用
       - 在 Property Editor 中确认 Avatar Opacity 不是 0

    2. **调整相机位置**:
       - 按 **“F”** 键聚焦到模型
       - 使用鼠标滚轮缩小视图

    3. **切换显示模式**:
       - 按 **“2”** 键切换到着色显示
       - 按 **“3”** 键切换到纹理显示

??? question "“模型太大 / 太小”"
    **处理方法**:

    1. **调整缩放**:
       - 在 Property Editor 中调整 Scale 数值
       - VRChat 模型通常在 0.8 到 1.2 之间

    2. **确认合适尺寸**:
       - 调整到模型身高约为 1.5m 到 2.0m
       - 同时考虑后续服装制作的操作便利性

??? question "“纹理（颜色 / 花纹）不显示”"
    **处理方法**:

    1. **切换显示模式**:
       - 按 **“3”** 键切换到纹理显示模式
       - 按 **“4”** 键切换到渲染显示模式

    2. **检查文件结构**:
       - 确认纹理文件与 FBX 文件位于相同位置
       - 确认该 FBX 是否为内嵌纹理的文件

    !!! info "关于纹理"
        这不会影响服装制作的基础流程，但在最终确认时会很重要。

## 📚 参考信息

### 关于模型文件

**FBX 文件中包含的信息**:
- **网格**: 3D 形状数据
- **骨骼**: 骨架信息（用于动画）
- **纹理**: 颜色与花纹信息
- **材质**: 材质信息

**VRChat 模型的特点**:
- 已针对 Unity 进行优化
- 骨骼布局为 Humanoid 形式
- 纹理符合 VRChat 的使用方式

### 项目管理技巧

**推荐的文件整理方式**:
```
VRChat衣装制作/
├── Avatars/
│   ├── MyAvatar_v1.fbx
│   └── MyAvatar_textures/ (有纹理时)
├── Projects/
│   └── MyAvatar_dress_project.zprj
└── Exports/
    └── (完成的服装文件)
```

## 📐 设计思路：为什么从 T 恤开始？

### T 恤适合作为起点的原因

!!! info "适合学习的原因"

    **1. 结构简单**
    - 只由基础几何形状构成
    - 没有复杂曲线或装饰
    - 版型结构容易理解

    **2. 失败风险低**
    - 修改起来更容易
    - 便于做实验性调整
    - 完成路径清晰

    **3. 应用范围广**
    - 是其他服装的基础
    - 便于扩展出不同变体
    - 能练到尺寸调整技术

## 🚀 步骤 1：设计与规划

### 理解基础设计

!!! example "理解设计图"

    **T 恤的组成部分**：
    1. **前身片（Front Panel）**：胸前部分
    2. **后身片（Back Panel）**：背部部分
    3. **袖子（左右各一）**：手臂部分
    4. **领口（Neckline）**：颈部周围

    **制作顺序**：
    1. 前身片 → 2. 后身片 → 3. 袖子 → 4. 缝合 → 5. 调整

### 尺寸设定基准

!!! example "根据头像设定尺寸"

    **基准测量值**（一般 VRChat 头像）：
    - **胸围**：80-90cm
    - **衣长**：60-70cm
    - **袖长**：25-30cm
    - **余量**：各部位约 5-10cm

    !!! tip "尺寸思路"
        一开始做得稍微大一点，后面再调整会更稳妥。

## 🛠️ 步骤 2：制作前身片

### 创建基础矩形版型

!!! example "步骤 2-1：创建前身片基础形状"

    **使用 Rectangle 工具创建基础形状**：
    1. 选择 **“R”** 键（或 Rectangle 工具）
    2. 在 2D Pattern Window 中按以下尺寸绘制矩形：
       - **宽度**：45cm（胸围 90cm 的一半）
       - **高度**：65cm（衣长）
    3. 切回 **“A”** 键（Arrow 工具）

    **在 Property Editor 中精确设置尺寸**：
    1. 选中创建好的矩形
    2. 在 Property Editor 中确认 **“Size”**
    3. 如有需要，可直接输入数值进行调整

!!! example "步骤 2-2：处理领口"

    **制作 Neckline（领口）**：
    1. 选中前身片版型
    2. 点击 **“Edit Pattern”** 工具
    3. 选中上边中央部分
    4. 向下凹入约 10cm，制作领口
    5. 用曲线工具调整成自然的颈部形状

    **理想领口参数**：
    - 宽度：15-20cm
    - 深度：8-12cm
    - 曲线自然顺滑

## 🛠️ 步骤 3：制作后身片

### 以前身片为基础高效制作

!!! example "步骤 3-1：复制前身片并调整"

    **高效制作后身片的方法**：
    1. 选中前身片版型
    2. 按 **“Ctrl + C”** 复制
    3. 按 **“Ctrl + V”** 粘贴
    4. 将新的版型移动到右侧（作为后身片）

    **后身片专属调整**：
    1. 将领口调浅（比前身片浅约 5cm）
    2. 整体长度设为比前身片长 2-3cm
    3. 将版型名称改为 **“Back”**（便于区分）

!!! tip "关于前后片的区别"
    - **前身片**：领口更深，衣长略短
    - **后身片**：领口更浅，衣长略长

## 🛠️ 步骤 4：制作袖子

### 袖子版型的基础结构

!!! example "步骤 4-1：袖子的基础形状"

    **创建袖子版型**：
    1. 使用 **Rectangle 工具** 创建两个相同尺寸的矩形：
       - **宽度**：40cm（包含袖口余量）
       - **高度**：28cm（袖长）
    2. 分别放在头像左右手臂附近

    **调整袖口与袖山连接处**：
    1. 使用 **Edit Pattern 工具** 调整袖子形状
    2. 将袖山连接部分（与肩膀相连的部分）稍微做成曲线
    3. 袖口（手腕侧）略微收窄

!!! example "步骤 4-2：袖子的摆放与方向"

    **正确的袖子摆放方式**：
    - **左袖**：放在头像左臂附近
    - **右袖**：放在头像右臂附近
    - **方向**：让袖山连接部分朝向头像肩部

## 🧵 步骤 5：设置缝合线（Internal Line）

### 指定需要缝合的部分

!!! example "步骤 5-1：绘制缝合线"

    **使用 Internal Line 工具**：
    1. 从工具栏选择 **“Internal Line”** 工具
    2. 按顺序绘制以下缝合线：

    **需要的缝合线**：
    1. **肩线**：连接前身片与后身片的肩部
    2. **侧缝线**：连接前身片与后身片的侧边
    3. **袖子连接线**：连接衣身与袖子的部分
    4. **袖下线**：封闭袖子下方的线

!!! warning "缝合线注意事项"
    - 使用红线（Internal Line）绘制
    - 调整到两个要连接的边长度一致
    - 线的起点和终点必须准确贴到版型边缘

!!! example "步骤 5-2：执行缝合"

    **执行自动缝合**：
    1. 确认所有缝合线都已正确绘制
    2. 点击 **“2D Pattern”** → **“Free Sew”**
    3. 如果报错，修正对应的缝合线
    4. 成功后会自动完成缝合

## 🎭 步骤 6：首次模拟与基础调整

### 执行模拟

!!! example "步骤 6-1：首次确认适配效果"

    **执行模拟**：
    1. 确认各版型已摆放在头像周围
    2. 按 **“空格键”** 开始模拟
    3. 观察 T 恤穿到头像上的过程
    4. 检查初始问题

    **常见初始状态**：
    - ✅ **理想情况**：贴合度适中
    - ⚠️ **需要调整**：过紧或过松
    - ❌ **需要修正**：尺寸差距过大或形状有明显问题

### 基础调整作业

!!! example "步骤 6-2：调整尺寸与贴合度"

    **整体尺寸调整**：
    1. 如果存在问题，先**停止模拟**
    2. 按 **“Ctrl + A”** 选中全部版型
    3. 通过 **“Transform”** → **“Scale”** 调整尺寸
    4. 再次模拟确认结果

    **局部调整**：
    - **胸部过紧**：加宽前身片
    - **衣长过长**：调整前后身片高度
    - **袖子过紧**：加宽袖子
    - **领口过窄**：扩大 Neckline

## 🎨 步骤 7：细节调整与质量提升

### 优化物理特性

!!! example "步骤 7-1：布料物理设置"

    **适合 VRChat 的物理设置**：
    1. 选中全部版型
    2. 在 Property Editor 中确认 **“Physical Property”**
    3. 应用以下设置：
       - **Fabric Preset**：Cotton（棉）
       - **Particle Distance**：8（兼顾质量与轻量化）
       - **Bend**：0.3（适中的柔软度）
       - **Stretch**：0.1（轻微弹性）

    **这些设置的意图**：
    - 优先保证在 VRChat 中稳定运行
    - 避免过于真实的物理导致负载过高
    - 保持自然观感

### 调整外观

!!! example "步骤 7-2：颜色与纹理基础设置"

    **设置基础颜色**：
    1. 选中版型并查看 Property Editor
    2. 在 **“Color”** 中设置喜欢的颜色
    3. 将前身片、后身片和袖子统一成同一颜色

    **推荐配色示例**：
    - **白色**：RGB(255, 255, 255) - 清爽
    - **海军蓝**：RGB(25, 25, 112) - 稳重
    - **灰色**：RGB(128, 128, 128) - 通用

## 🔧 故障排查

### T 恤制作中常见的问题

??? question "“缝合线报错”"
    **处理方法**：

    1. **确认线段长度**：
       - 两条要连接的边长度差异过大
       - 用 Edit Pattern 调整长度

    2. **确认线的位置**：
       - Internal Line 的起点和终点没有准确贴到版型边缘
       - 放大视图后重新精确绘制

    3. **形状过于复杂**：
       - 先从简单直线开始
       - 熟练后再增加曲线

??? question "“T 恤形状变得很奇怪”"
    **处理方法**：

    1. **重新检查版型尺寸**：
       - 回到基础尺寸（约 45×65cm）
       - 逐步调整

    2. **确认缝合顺序**：
       - 按肩部 → 侧边 → 袖山连接 → 袖下的顺序
       - 不要一次性全部缝完，分阶段进行

    3. **重置物理设置**：
       - 将 Physical Property 恢复默认
       - 从 Cotton 预设重新开始

??? question "“尺寸调整总是调不好”"
    **处理方法**：

    1. **利用局部调整**：
       - 不要只靠整体 Scale 调整，改用单独版型处理
       - 尤其是胸部和腰部，局部调整更有效

    2. **确认头像尺寸**：
       - 检查头像本身的缩放是否处于常规范围
       - 必要时也考虑调整头像的 Scale

    3. **参考标准值**：
       - 参考常见 T 恤尺寸
       - 一开始先做大一点，再往回缩会更好调

## 🌟 Level 1: 简单喇叭裙

### 设计与规划

!!! example "步骤 1-1：喇叭裙设计"

    **确定基础规格**：
    ```
    裙长: 膝上 5cm（40-45cm）
    腰围: 70-75cm（含余量）
    下摆周长: 120-140cm（展开感）
    材质感: 轻盈的棉质系
    ```

    **理解喇叭形状**：
    - **腰部**：较窄的圆弧
    - **下摆**：较宽的圆弧
    - **侧面**：用平缓曲线连接
    - **前后**：基本保持左右对称

### 创建版型

!!! example "步骤 1-2：创建基础版型"

    **基于圆弧的版型创建**：
    1. 选择 **Polygon 工具**
    2. 按以下尺寸制作接近梯形的形状：
       - **腰部**：宽 35cm（半圈）
       - **下摆**：宽 60cm
       - **长度**：45cm
    3. 使用 **Edit Pattern** 调整为曲线

    **调整腰线**：
    1. 选中腰部边线
    2. 使用 **Curve Tool** 调整为自然曲线
    3. 让它贴合模型的腰线形状

    **调整下摆线**：
    1. 选中下摆部分
    2. 调成能表现轻盈展开感的曲线
    3. 前后长度略作变化，让效果更自然

!!! example "步骤 1-3：创建前后版型"

    **创建后片**：
    1. 复制前片（**Ctrl + C**、**Ctrl + V**）
    2. 对后片稍作调整：
       - 长度比前片长 2-3cm
       - 腰线略微提高
    3. 将版型放在右侧（后方）

### 缝合与模拟

!!! example "步骤 1-4：设置缝合线"

    **创建 Side Seam（侧缝）**：
    1. 选择 **Internal Line** 工具
    2. 连接前片与后片的侧边
    3. 左右两侧都创建 Side Seam

    **处理腰线**：
    1. 腰部先保持开放（腰带进阶再加）
    2. 如有需要，可用 **Hem** 工具处理边缘

!!! example "步骤 1-5：物理设置与模拟"

    **喇叭裙专用物理设置**：
    ```
    Fabric Preset: Cotton
    Particle Distance: 6-8（为了更优美的动作）
    Bend: 0.1-0.2（轻盈摆动）
    Stretch: 0.05-0.1（适度张力）
    Air Resistance: 0.1-0.2（受风摆动）
    Density: 0.15-0.2（轻盈感）
    ```

    **首次执行模拟**：
    1. 将版型摆放在模型腰部周围
    2. 按 **空格键** 开始模拟
    3. 观察裙子的展开与下垂效果

## 🔄 Level 2: 百褶裙

### 理解百褶与设计方法

!!! example "步骤 2-1：理解百褶理论"

    **百褶的基础结构**：
    ```
    褶距: 3-4cm（折返间距）
    褶深: 1.5-2cm（折入深度）
    褶数: 16-20 个（围绕腰部）
    总布宽: 腰围的 2.5-3 倍（考虑褶量）
    ```

    **百褶版型计算**：
    - **显示宽度**：70cm（腰围）
    - **实际布宽**：180-200cm（展开后的百褶宽度）
    - **每个褶子**：宽 10-12cm

### 创建百褶版型

!!! example "步骤 2-2：创建百褶版型"

    **创建基础形状**：
    1. 用 **Rectangle** 工具创建大矩形：
       - **宽度**：90cm（半圈展开宽度）
       - **高度**：40cm（裙长）
    2. 创建两份（前、后）

    **制作褶线标记**：
    1. 用 **Line** 工具标记百褶线
    2. 每隔 3cm 画一条竖线
    3. 用 **Fold** 工具指定百褶折叠方向

    **执行百褶折叠**：
    1. 使用 **Pleat** 工具（或 Fold 功能）
    2. 交替设置山折与谷折
    3. 统一百褶深度

!!! example "步骤 2-3：固定百褶并模拟"

    **百褶保持设置**：
    ```
    Fabric Preset: Cotton（稍微偏硬）
    Bend: 0.3-0.5（保持百褶形状）
    Stretch: 0.2-0.3（稳定性）
    Shear: 0.2-0.3（保持结构）
    Particle Distance: 8-10（稳定性与质量平衡）
    ```

    **分阶段模拟**：
    1. 先执行百褶折叠的模拟
    2. 等百褶稳定后再放到模型上
    3. 做最终调整并确认贴合度

## 🎭 Level 3: 分层裙

### 设计多层结构

!!! example "步骤 3-1：规划层次结构"

    **三层结构示例**：
    ```
    Layer 1（最下层）: 长裙（膝下 5cm）
    Layer 2（中层）: 中长裙（膝上 5cm）
    Layer 3（最上层）: 短裙（大腿中部）
    ```

    **各层特点**：
    - **最下层**：偏重，动作稳定
    - **中层**：轻盈感适中
    - **最上层**：最轻，摆动更明显

### 制作各个单独图层

!!! example "步骤 3-2：制作各层"

    **最下层（基础裙）**：
    ```
    长度: 55cm
    展开程度: 较克制
    物理设置: Density 0.3, Bend 0.4
    颜色: 深色系（基础色）
    ```

    **中层（中间裙）**：
    ```
    长度: 35cm
    展开程度: 中等
    物理设置: Density 0.2, Bend 0.3
    颜色: 主色
    ```

    **最上层（顶部裙）**：
    ```
    长度: 25cm
    展开程度: 更大
    物理设置: Density 0.15, Bend 0.2
    颜色: 点缀色或图案
    ```

### 整合图层并调整

!!! example "步骤 3-3：多层模拟"

    **调整层间间距**：
    1. 微调各层腰部位置
    2. 最上层设得最高，下层逐步降低
    3. 保持约 1-2cm 间距

    **整合模拟设置**：
    ```
    Self Collision: 仅在层间启用
    Collision Thickness: 2-3mm（保持层间分离）
    Air Resistance: 各层做出差异
    ```

    **VRChat 优化注意点**：
    - 调整总多边形数量，确保不超出标准
    - 省略不必要的细节
    - 通过 Particle Distance 进行轻量化

## ⚙️ 裙子特有的物理设置

### 为了优美动作而做的优化

!!! example "裙子物理设置最佳实践"

    **喇叭裙最佳设置**：
    ```
    Particle Distance: 6-8
    Bend: 0.1-0.3（轻盈摆动）
    Air Resistance: 0.1-0.2（受风影响）
    Density: 0.15-0.25（轻盈）
    Self Collision: 仅下摆启用
    ```

    **百褶裙最佳设置**：
    ```
    Particle Distance: 8-10
    Bend: 0.3-0.5（保持百褶）
    Stretch: 0.2-0.3（形状稳定）
    Shear: 0.2-0.3（保持褶线）
    Density: 0.2-0.3（适中的重量感）
    ```

### 针对动作的调整技巧

!!! example "根据模型动作进行调整"

    **步行时的调整**：
    - 提高 Air Resistance，形成自然摆动
    - 降低 Bend，让它更好跟随腿部动作

    **坐姿的对应**：
    - 启用 Self Collision 防止穿透
    - 通过适度 Stretch 让膝部附近更贴合

    **舞蹈或激烈动作的对应**：
    - 提高 Density 以保证稳定性
    - 通过 Damping 功能抑制振动

## 🎨 装饰与变体

### 自定义基础裙子

!!! example "展开设计变体"

    **下摆装饰**：
    1. 用 **Hem Tool** 做蕾丝风包边
    2. 用 **Ruffle** 添加波浪装饰
    3. 做出带有高低差的不对称下摆

    **添加腰带**：
    1. 单独创建腰带版型
    2. 制作宽 5-8cm 的带状版型
    3. 与主裙进行缝合

    **颜色与材质变体**：
    - **纯色**：单色简洁风格
    - **图案**：格纹、条纹、花纹
    - **渐变**：颜色逐级变化
    - **金属感**：带光泽的特殊材质感

### 分层风格的进阶

!!! example "高级分层技法"

    **不对称分层**：
    - 左右长度不同的层次
    - 前后结构不同
    - 与斜切组合

    **材质混搭**：
    - 带透明感的图层
    - 不同物理特性的组合
    - 按材质区分表现效果

## 📤 Unity 集成与 VRChat 优化

### 裙子特有的优化

!!! example "裙子的 VRChat 优化"

    **多边形数量管理**：
    ```
    简单喇叭裙: 3,000-5,000 triangles
    百褶裙: 5,000-8,000 triangles
    分层裙: 8,000-12,000 triangles（需注意）
    ```

    **材质优化**：
    - 层与层之间共用材质
    - 利用 UV Map 做颜色区分
    - 优化透明度设置

    **物理优化**：
    - 面向 VRChat 调整 Particle Distance
    - 将 Self Collision 控制在最小范围
    - 移除不必要的高精度设置

## 💡 创意灵感

### 设计提示

**季节感的表现**：
- **春季**：轻盈喇叭轮廓、粉彩配色
- **夏季**：带透明感的分层、清爽动作感
- **秋季**：厚重百褶、深色调
- **冬季**：多层叠穿、厚实材质感

**文化与风格表达**：
- **和风**：袴风百褶结构
- **西式**：经典喇叭轮廓
- **现代风**：几何式不对称
- **奇幻风**：仿佛违背重力的轻盈感

请以这些创意为参考，制作属于你自己的原创裙子。

## 🎯 上传流程概览

### VRChat 上传机制

!!! info "上传全貌"

    **本地测试 → 上传 → 公开确认**

    **1. 最终确认阶段**
    - 在 Unity 中做最后检查
    - 确认 Performance 标准
    - 设置头像信息

    **2. 上传阶段**
    - 使用 VRChat SDK Build & Publish
    - 设置缩略图
    - 选择公开范围

    **3. 确认阶段**
    - 在 VRChat 中测试运行
    - 从他人视角确认外观
    - 如有问题则修正并重新上传

## 🔍 步骤 1: 最终质量检查

### 最终确认头像状态

!!! example "步骤 1-1: 在 Unity 中做最后检查"

    **最终确认外观**:
    1. 在 **Scene View** 中 360 度查看头像
    2. 确认服装贴合良好
    3. 确认材质和纹理显示正确
    4. 确认没有不自然的穿插或悬空

    **确认 Animation**:
    1. 在 **Scene View** 中选择头像
    2. 在 **Inspector** → **Animator** 中确认状态
    3. 确认基础姿势下没有不自然变形

!!! example "步骤 1-2: 使用 VRChat SDK 评估性能"

    **确认 Performance Rating**:
    1. **“VRChat SDK”** → **“Show Control Panel”**
    2. 在 **“Builder”** 标签页中选择头像
    3. 确认 **Performance Rating**：

    **PC 版标准**:
    - **Excellent**: 绿色 - 所有项目都在推荐范围内
    - **Good**: 黄色 - 基本都在推荐范围内
    - **Medium**: 橙色 - 稍重
    - **Poor**: 红色 - 需要优化

    **如果要支持 Quest 版**:
    - 标准会更严格（例如多边形数量需低于 7,500）
    - 如果不需要支持 Quest，可按 PC 版标准判断

!!! warning "关于 Performance 评估"
    即使是 “Poor” 仍然可以上传，但其他用户有可能无法看到你的头像。建议尽量达到 “Good” 以上。

### 最终调整头像设置

!!! example "步骤 1-3: Avatar Descriptor 最终设置"

    **确认重要设置**:
    1. **View Position** 是否设置正确
       - 绿色球位于头像眼睛位置
       - 高度一般约为 1.6～1.8m
    2. **Lip Sync** 是否已设置
       - Jaw Bone 已被正确选中
    3. **Eye Look** 设置（可选）
       - 如果希望眼球运动更自然

    **动画设置**:
    - **Playable Layers** 保持默认即可
    - 如果不需要特殊动作，通常无需修改

## 📤 步骤 2: 上传到 VRChat

### 执行 Build & Publish

!!! example "步骤 2-1: 开始上传"

    **执行 Build & Publish**:
    1. 在 VRChat SDK Control Panel 中打开 **“Builder”** 标签页
    2. 确认选中了要上传的头像
    3. 点击 **“Build & Publish for Windows”**
    4. 构建流程开始运行（约 2-5 分钟）

    **构建过程中的确认事项**:
    - 查看 Console Window 是否出现错误
    - 确认进度条是否正常推进
    - 确认 Unity 没有卡死

!!! tip "关于构建时间"
    首次构建会比较耗时，第二次以后通常会更快，请耐心等待。

### 设置头像信息

!!! example "步骤 2-2: 输入上传信息"

    **构建完成后会显示设置界面**:

    **基础信息**:
    1. **Avatar Name**: 输入便于识别的名称
       - 例如: `MyAvatar_Tshirt`、`カジュアル衣装_v1`
    2. **Description**: 输入简短说明
       - 例如: “穿着手工 T 恤的头像”

    **缩略图**:
    1. 点击 **“Take Picture”**
    2. 调整到能清楚展示头像的角度
    3. 拍摄图片（会作为 VRChat 内显示的图标）
    4. 如果不满意，可点击 **“Retake”** 重拍

!!! example "步骤 2-3: 公开设置与最终确认"

    **设置可见范围**:
    1. **“Visibility”** 设置：
       - **Private**: 仅自己可用（推荐首次使用）
       - **Friends**: 好友也可使用
       - **Public**: 任何人都可使用

    **最终确认**:
    1. 确认头像名称和说明无误
    2. 确认缩略图合适
    3. 点击 **“Upload”**

!!! warning "首次建议选择 Private"
    建议第一次先用 “Private” 上传，确认运行正常后再逐步扩大公开范围。

### 上传完成与确认

!!! example "步骤 2-4: 确认上传完成"

    **确认上传成功**:
    1. 会显示上传完成的消息
    2. VRChat 网站上会新增该头像
    3. 可在 VRChat 应用中选择这个新头像

    **上传失败时**:
    - 确认错误消息
    - 常见原因：网络连接、账号权限
    - 稍等一段时间后重试

## 🎮 步骤 3: 在 VRChat 中测试运行

### 在 VRChat 中选择头像

!!! example "步骤 3-1: 在 VRChat 应用中确认"

    **启动 VRChat 并确认头像**:
    1. 启动 **VRChat** 应用
    2. 登录后打开 **“Avatars”** 菜单
    3. 在 **“Mine”** 标签页中确认自己的头像列表
    4. 确认能看到刚上传的新头像

    **选择头像**:
    1. 点击刚上传的新头像
    2. 通过 **“Change to Avatar”** 穿上它
    3. 确认头像已经切换

!!! success "头像穿戴成功！"
    到这一步，你已经能在 VRChat 中实际穿上自己制作的服装了。

### 确认基础动作

!!! example "步骤 3-2: 测试基础动作"

    **从自己视角确认**:
    1. **步行**: 确认移动时服装的动作
    2. **手部动作**: 抬手或挥手时确认表现
    3. **跳跃**: 确认跳跃时服装的行为
    4. **坐下**: 确认坐在椅子等物体上时的状态

    **确认表情与动作**:
    1. 用 **Expression Menu** 切换表情
    2. 用 **Emote** 试试跳舞或摆姿势
    3. 确认服装没有出现不自然的动作

!!! example "步骤 3-3: 通过镜子确认外观"

    **用镜子检查外观**:
    1. 前往 VRChat 中有镜子的地方
    2. 通过镜子从不同角度确认自己的头像：
       - 正面、侧面、背面的外观
       - 服装贴合程度
       - 颜色与质感的显示效果
       - 是否有不自然的部分

    **截图留念**:
    1. 打开 **“Camera”** 菜单
    2. 以合适的角度拍摄截图
    3. 作为制作过程记录保存

## 🔧 步骤 5: 处理问题与提升质量

### 处理常见问题

!!! example "步骤 5-1: 处理发现的问题"

    **如果服装动作不自然**:
    1. 回到 Unity 调整物理设置
    2. 在 Marvelous Designer 中重新检查布料设置
    3. 重新导出 → Unity 集成 → 重新上传

    **如果外观有问题**:
    1. 在 Unity 中调整材质设置
    2. 修正纹理质量或颜色
    3. 如有需要，在 Marvelous Designer 中调整颜色

    **如果有性能问题**:
    1. 考虑减少多边形数量
    2. 优化纹理尺寸
    3. 合并不必要的材质

### 版本管理与改进

!!! example "步骤 5-2: 持续改进"

    **版本管理实践**:
    1. **记录问题点**: 列出发现的问题
    2. **制作改进版**: 在 Unity/Marvelous Designer 中修正
    3. **上传新版本**: 以 “v2” 等形式创建新头像

    **改进循环**:
    ```
    制作 → 测试 → 发现问题 → 修正 → 再测试 → 完成
    ```

## 🔧 持续性的故障排查

### 长期使用时的注意点

??? question "“上传的头像看不到”"
    **处理方法**:

    1. **确认 Avatar Display 设置**:
       - 在 VRChat 设置中确认他人头像显示相关设置
       - 确认是否因 Performance 设置而被隐藏

    2. **重新选择头像**:
       - 先切换到其他头像，再重新选择当前头像
       - 重启 VRChat 应用

??? question "“服装有时显示异常”"
    **处理方法**:

    1. **网络问题**:
       - 确认互联网连接
       - 确认 VRChat 服务器状态

    2. **清理缓存**:
       - 清理 VRChat 缓存
       - 重新下载头像数据

??? question "“我想上传新版本”"
    **处理方法**:

    1. **更新现有头像**:
       - 使用相同名称上传时会覆盖更新
       - 使用新名称则会作为新头像新增

    2. **版本管理**:
       - 在头像名称中加入 “v1”“v2”等版本号
       - 删除旧版本或将其设为非公开

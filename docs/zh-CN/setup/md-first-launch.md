# Marvelous Designer 首次启动指南

!!! note "指南信息"
    **预计用时**：约 30 分钟 | **难度**：适合初学者

**本页目标**：首次启动 Marvelous Designer，理解基础设置与界面结构。

## 🚀 首次启动步骤

### 步骤 1：启动 Marvelous Designer

!!! example "步骤 1-1：先启动看看"

    1. 双击桌面上的 **“Marvelous Designer”** 图标
    2. 或在开始菜单中搜索 “Marvelous Designer” 并启动

    **启动时可能看到的内容**：

    - ✅ **许可证认证界面**：仅首次显示
    - ✅ **启动画面**：稍等片刻即可
    - ✅ **教程推荐**：可以先跳过，之后再看

!!! tip "如果启动较慢"
    首次启动可能会稍慢一些。先等待 1 到 2 分钟再判断是否异常。

### 步骤 2：初始设置

!!! example "步骤 2-1：确认语言设置"

    启动 Marvelous Designer 后，先确认语言设置。

    **将语言切换为日文的方法**：

    1. 点击顶部菜单 **“Edit”**
    2. 选择 **“Preferences”**（或 **“环境设置”**）
    3. 在左侧列表中选择 **“General”**
    4. 在 **“Language”** 的下拉菜单中选择 **“日本語”**
    5. 点击 **“OK”**
    6. 如果提示重启，请按提示重启

!!! warning "关于日文界面"
    某些版本可能没有日文菜单。此时可以继续使用英文界面；本指南会尽量同时写出两种菜单名称。

!!! example "调整基础设置"

    **建议先把这些设置优化好**：

    **自动保存**：
    1. **“Edit”** → **“Preferences”** → **“General”**
    2. 启用 **“Auto Save”**
    3. 将 **“Auto Save Interval”** 设为 5 分钟

    **单位设置**：
    1. **“Edit”** → **“Preferences”** → **“General”**
    2. 将 **“Units”** 设为 **“Metric(cm)”**

    **内存/GPU 设置**：
    1. **“Edit”** → **“Preferences”** → **“3D Graphics”**
    2. 启用 **“Use GPU”**（若你的电脑有独立显卡）

    !!! success "设置完成"
        设置完成后，记得点击 **“Apply”** → **“OK”** 保存。

### 步骤 3：理解界面结构

!!! example "主界面由哪些部分组成"

    Marvelous Designer 的主界面大致分为 4 个区域：

    **📐 2D 版型窗口（左侧）**：
    - 用于创建和编辑服装版型
    - 这里进行平面操作
    - 通常显示为 **“2D Pattern Window”**

    **🎭 3D 视口（右侧）**：
    - 用于查看 3D 头像与服装
    - 可以实时确认穿着效果
    - 通常显示为 **“3D Garment Window”**

    **🛠️ 工具栏（上方）**：
    - 集中放置常用功能按钮
    - 包括版型创建、选择、移动等基础工具

    **📊 属性面板（右侧或下方）**：
    - 用于设置所选对象的详细属性
    - 包括物理特性与模拟设置

    !!! tip "如果界面太小"
        可以拖动各面板之间的分隔线，调整成自己更容易观察的尺寸。

!!! example "记住基础视角操作"

    **3D 视口的操作方法**：

    - **旋转**：按住鼠标右键拖动
    - **平移**：按住鼠标中键拖动
    - **缩放**：滚动鼠标滚轮
    - **聚焦显示**：按 **“F”** 键

    **实际试一下**：
    1. 在 3D 视口中按住右键并移动鼠标
    2. 用滚轮放大/缩小
    3. 按 **“F”** 键重置视图

    !!! success "你已经熟悉基本操作了"
        这些操作之后会一直用到，先稍微练一下会省很多时间。

### 步骤 4：创建第一个项目

!!! example "创建一个新项目"

    **创建步骤**：
    1. 点击 **“File”** → **“New Project”**
    2. 输入项目名：**“练习用项目”**
    3. 设置保存位置：放在刚才创建的 **“VRChat服装制作”** 文件夹中
    4. 点击 **“Create”**

    **创建后的界面状态**：
    - 3D 视口中会显示一个基础人形（头像）
    - 2D 版型窗口为空
    - 这就是正常的新项目初始状态

    !!! info "关于默认头像"
        初始显示的是通用人形模型，后面会替换成你的 VRChat 头像（FBX 文件）。

### 步骤 5：测试基础功能

!!! example "试着确认软件是否正常工作"

    **先创建一个简单的矩形**：

    1. 在左上角工具栏中点击 **“Rectangle”** 工具（矩形图标）
    2. 在 2D 版型窗口中点击并拖动，画出一个矩形
    3. 创建完成后切回普通工具（点击 **“Arrow”** 工具）

    **执行模拟**：
    1. 点击 **“Simulation”** → **“Simulate”**
    2. 或直接按 **空格键**
    3. 你会看到创建的矩形出现在 3D 视口里，并在重力作用下下落

    **停止模拟**：
    1. 再次按 **空格键**
    2. 或点击 **“Simulation”** → **“Reset Simulation”**

    !!! success "恭喜"
        你已经在 Marvelous Designer 中创建了第一块“布料”，并成功运行了物理模拟。

## 🎨 界面自定义

!!! example "调整成更顺手的工作环境"

    **调整面板布局**：

    1. **调整窗口大小**：
       - 拖动各面板之间的边界，调整 2D/3D 的显示比例
       - 通常把 3D 视口稍微放大一些会更好用

    2. **工具栏位置**：
       - 确认常用工具是否在容易看到的位置
       - 必要时可通过 **“View”** → **“Toolbars”** 调整

    3. **颜色设置**：
       - **“Edit”** → **“Preferences”** → **“Display”**
       - 调整背景颜色、网格显示等，让界面更容易观察

    !!! tip "保存你自己的布局"
        如果你调到了满意的配置，可以通过 **“File”** → **“Save Layout”** 保存。

## 📚 理解基础概念

!!! example "Marvelous Designer 的核心思路"

    **需要先理解的关键概念**：

    **1. 版型（Pattern）**：
    - 就像现实制衣中的“纸样”
    - 先在 2D 中创建，再在 3D 中形成服装
    - 可以从矩形、圆形等基础形状逐步做出复杂版型

    **2. 模拟（Simulation）**：
    - 用于计算布料物理运动的功能
    - 会考虑重力、风、碰撞等因素
    - 可以实时查看结果

    **3. 头像（Avatar）**：
    - 用来穿衣的人形模型
    - 后续会替换成你的 VRChat 头像
    - 它是确认服装贴合度的重要对象

    !!! info "与现实制衣的关系"
        Marvelous Designer 本质上是把现实制衣流程数字化。版型制作 → 缝合 → 适配，这条流程并没有变。

## 🔧 常见问题

??? question "“模拟没有反应”"
    **处理方法**：
    1. 确认已经创建版型
    2. 在 **“Simulation”** → **“Simulation Properties”** 中确认 Gravity（重力）已启用
    3. 尝试降低图形设置

??? question "“界面错乱，或者有一部分看不见”"
    **处理方法**：
    1. 通过 **“Window”** → **“Reset Layout”** 恢复默认布局
    2. 确认显示器分辨率和缩放设置
    3. 把显卡驱动更新到最新版本

??? question "“操作总是不顺手”"
    **处理方法**：
    1. 按 **“F1”** 键打开帮助
    2. 用 **“Ctrl + Z”** 撤销操作
    3. 通过 **“File”** → **“Recent Projects”** 重新打开最近项目

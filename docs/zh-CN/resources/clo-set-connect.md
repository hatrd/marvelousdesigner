# CLO-SET Connect 与 EveryWear 平台

## 什么是 CLO-SET Connect

**CLO-SET Connect** 是面向 3D 时尚设计的全球性**数字市场平台**。对于 VRChat 创作者来说，这是一个能够轻松获取高质量 3D 服装素材和工具的重要平台。

!!! info "平台概览"
    - **URL**: [https://connect.clo-set.com/](https://connect.clo-set.com/)
    - **语言**: 支持英语和日语
    - **费用**: 免费账号 + 付费素材
    - **支持软件**: CLO、Marvelous Designer、Blender 等

## 🌟 主要功能与优势

## EveryWear - 面向游戏与元宇宙的优化

**EveryWear** 是一套特殊功能包，可将使用 CLO 和 Marvelous Designer 制作的服装**优化为适用于游戏和元宇宙的形式**。

### 🎮 主要功能

#### 1. **自动适配**
```text
功能：自动将 2D 版型和 3D 服装调整为不同模型尺寸
用途：将同一套服装应用到不同体型的 VRChat 模型
效果：大幅缩短手动调整时间
```

#### 2. **自动绑定骨架**
```text
功能：将 3D 服装装配到角色上，并定义实时动作表现
用途：让服装在 VRChat 中自然运动
效果：自动化复杂的绑定骨架工作
```

#### 3. **纹理烘焙**
```text
功能：将高分辨率 3D 服装的信息转写到 2D 纹理贴图
用途：制作适用于 VRChat 的轻量化纹理
效果：在保持质量的同时减小文件体积
```

### 在 VRChat 制作中的使用示例

#### 面向初学者：利用现成素材
```text
1. 在 Connect 上购买喜欢的连衣裙 PZIP 文件
2. 使用 EveryWear 自动适配到模型
3. 应用自动绑定骨架后完成 VRChat 适配
→ 制作时间：1-2 小时
```

#### 面向中级用户：定制化制作
```text
1. 在 Connect 上购买基础素材和面料
2. 在 Marvelous Designer 中按自己的需求修改
3. 使用 EveryWear 进行优化处理
4. 在 Unity 中集成并上传到 VRChat
→ 制作时间：4-6 小时
```

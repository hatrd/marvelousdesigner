## 🔧 常见问题与解决方案

### Q1: EveryWear 功能无法运行

??? question "“按下 Auto-Fit 按钮后没有反应”"
    **检查项**:

    **1. 确认 CLO-SET Connect 联动**
    ```
    Tools → EveryWear → Account Status
    → 确认登录状态
    → 确认订阅有效
    ```

    **2. 确认头像文件格式**
    ```
    支持格式: FBX, OBJ
    推荐: VRChat 标准 FBX 格式
    骨骼结构: 必须兼容 Humanoid
    ```

    **3. 再次确认系统要求**
    ```
    网络连接: 必需
    防火墙: 允许 EveryWear 通信
    杀毒软件: 可暂时禁用测试
    ```

### Q2: AI 生成结果质量较低

??? question "“AI Pattern Generation 的结果和预期不符”"
    **提升质量的方法**:

    **优化参考图**:
    ```
    分辨率: 1024x1024 以上
    角度: 正面、侧面、背面三张一组
    背景: 推荐纯色背景
    光照: 均匀且明亮
    ```

    **优化提示词**:
    ```
    具体描述: “夏季连衣裙、及膝、无袖”
    风格说明: “面向 VRChat、简洁设计”
    排除项: “不需要复杂装饰”
    ```

### Q3: IK-Joint 联动不正常

??? question "“服装动作不自然”"
    **设置确认项**:

    **头像骨骼结构**:
    ```
    1. 确认 Humanoid 骨骼布局
    2. 统一命名规则（Left/Right 表记）
    3. 检查骨骼层级结构是否一致
    ```

    **MD2025 设置**:
    ```
    Tools → IK Settings →
    ✓ Auto Joint Detection: Enabled
    ✓ Bone Mapping: VRChat Standard
    ✓ Weight Transfer: Automatic
    ```

### Q4: 使用 UV Extension 后出现纹理变形

??? question "“编辑 UV 后纹理变形了”"
    **处理步骤**:

    **准备工作**:
    ```
    1. 备份原始 UV 布局
    2. 准备高分辨率纹理文件
    3. 提前确认接缝位置
    ```

    **设置调整**:
    ```
    UV Extension Settings →
    Distortion Tolerance: Conservative
    Seam Detection: Manual Override
    Packing Efficiency: Medium（优先质量）
    ```

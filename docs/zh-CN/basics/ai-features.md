# Marvelous Designer AI 功能指南

## AI 功能概述

自 Marvelous Designer 2024.1 起引入的 **AI 功能**，是能够大幅提升纹理制作与图形生成效率的创新工具。它的设计目标是让初学者也能轻松制作出美观的服装设计。

!!! info "支持版本"
    - 可在 **Marvelous Designer 2024.1 及以上版本** 中使用
    - 由于属于 **Beta 功能**，后续更新中规格可能发生变化
    - 当前 **不支持日语提示词**（仅支持英文）

## 可用的 AI 功能

### 🎨 AI 纹理生成器（Beta）

只需输入关键词，就能自动生成各种布料纹理的功能。

**主要特点：**
- **关键词输入**: 使用 `leather`、`denim`、`silk` 等英文单词生成纹理
- **实时预览**: 可立即在 3D Garment 上确认效果
- **无缝纹理**: 自动生成没有接缝的精美纹理
- **参数调整**: 可调整布料类型、内容类型、版型尺寸

**使用方法：**

1. **选择 Fabric**
   ```text
   服装 > Fabric > 选择想使用的 Fabric
   ```

2. **进入 AI 功能**
   ```text
   Fabric 属性窗口 > 基本参数按钮旁边
   > 点击“AI 纹理生成器（Beta）”按钮
   ```

3. **生成纹理**
   ```text
   - 在对话框顶部设置选项
   - 在文本提示词栏中输入英文关键词
   - 生成的图像会显示在底部
   ```

**提示词示例：**
```text
cotton fabric        # 棉布面料
leather texture      # 皮革纹理
silk pattern        # 丝绸图案
denim material      # 牛仔布材质
knit wool          # 羊毛针织
```

### 🖼 AI 图形生成器（Beta）

可将图形图像轻松应用到版型上的功能。

**主要特点：**
- **关键词生成**: 通过文本输入生成多种图形
- **版型应用**: 生成后的图形可直接应用到版型
- **历史记录功能**: 可从历史记录中重复使用已生成的图形
- **变形调整**: 可自由调整旋转、缩放与位置

**使用方法：**

1. **准备版型**
   ```text
   2D 版型窗口 > 选择要应用图形的版型
   ```

2. **调用 AI 功能**
   ```text
   右键点击版型 > 选择“AI 图形生成器”
   ```

3. **生成图形**
   ```text
   - 设置内容类型
   - 输入文本提示词（英文）
   - 从生成的图形中进行选择
   ```

**提示词示例：**
```text
floral pattern      # 花卉图案
geometric design    # 几何图案
logo print         # Logo 印花
stripe pattern     # 条纹图案
star design        # 星形图案设计
```

### 🏗 PBR Map 生成器

可根据纹理图像自动生成 PBR Map 的功能。

**可生成的内容：**
- **Normal Map**: 表现表面凹凸
- **Displacement Map**: 表现立体形状变化
- **Opacity Map**: 控制透明度
- **Roughness Map**: 表现表面粗糙度
- **Metalness Map**: 表现金属质感

**适用位置：**
- Trim（包边）
- Button Hole
- Stitch（缝线）
- 服装的褶皱与起皱

## 注意事项与最佳实践

### ⚠️ 重要限制

**语言限制：**
- 当前 **仅支持英文提示词**
- 输入日文时可能无法正常工作

**生成时间：**
- 在使用人数较多的时段，**生成可能耗时较长**
- 建议在工作日白天或深夜使用

**历史记录管理：**
- **关闭 Marvelous Designer 后，历史记录会被清除**
- 喜欢的纹理请 **立即保存**

**Beta 版注意事项：**
- 后续更新中，**功能或操作方式可能发生变化**
- 用于重要项目时，**请务必先做备份**

### 💡 高效使用技巧

**编写提示词：**
```text
具体: "red cotton" ✅ "fabric" ❌
补充形容词: "soft velvet texture" ✅ "velvet" △
多个关键词: "denim blue distressed vintage" ✅
```

**选择纹理：**
- 从多个候选中 **挑选最合适的结果**
- **细调可在之后** 通过 PBR 生成器完成
- 先 **测试应用**，再正式使用

**工作流集成：**
1. 用 AI 生成基础纹理
2. 手动调整细节
3. 用 PBR Map 提升质感
4. 做最终微调

## 故障排查

### 常见问题与处理方法

??? question "“纹理无法生成”"
    **检查项：**
    - Fabric 类型是否设置正确
    - 文本提示词是否以英文输入
    - 网络连接是否稳定

    **处理方法：**
    - 间隔一段时间后重试
    - 改用更简洁的提示词
    - 重启软件

??? question "“生成出的纹理质量较低”"
    **改进方法：**
    - 使用更具体的提示词
    - 尝试不同的 Fabric 类型
    - 调整生成选项中的参数

??? question "“历史记录中的纹理消失了”"
    **原因：**
    - 关闭 Marvelous Designer 后，历史记录会被自动删除

    **预防方法：**
    - 喜欢的纹理立即保存
    - 定期保存项目

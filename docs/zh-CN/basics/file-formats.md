# 文件格式的理解

!!! note "指南信息"
    **预计用时**: 约15分钟 | **难度**: 适合初学者 | **重要度**: 必学

**本页目的**: 理解 VRChat 服装制作中会用到的各类文件格式，并能根据场景正确区分使用。

## 📁 主要文件格式

### Marvelous Designer 相关

| 格式 | 说明 | 用途 |
|------|------|------|
| **.zprj** | 项目文件 | 保存与继续作业 |
| **.pzip** | 服装包 | 服装分发与共享 |
| **.fbx** | 3D 模型 | Unity 集成 |

### 纹理相关

| 格式 | 特点 | 推荐用途 |
|------|------|----------|
| **.png** | 支持透明度 | UI、含 Alpha 的图像 |
| **.jpg** | 体积小、画质高 | 基础纹理 |
| **.tga** | 高质量 | 制作用、中间格式 |

## 🔧 实用区分方式

### 在制作阶段中的使用方式
- **作业中**: 用 `.zprj` 保存
- **共享时**: 用 `.pzip` 分发
- **用于 Unity**: 导出为 `.fbx`

### VRChat 优化
- 纹理: 1024x1024 以下
- 多边形: 20,000 以下
- 文件大小: 50MB 以下

# 制作流程全貌

!!! note "指南信息"
    **预计用时**: 约20分钟 | **难度**: 适合初学者 | **重要度**: 必学（理解流程）

**本页目的**: 理解 VRChat 服装制作的整体流程，并掌握制定高效制作计划所需的知识。

!!! success "本指南可以学到什么"
    - ✅ **制作流程的整体脉络**
    - ✅ **各阶段的目的与重要性**
    - ✅ **时间分配与优先级**
    - ✅ **规避常见瓶颈的方法**
    - ✅ **质量管理要点**

!!! info "准备事项"
    - 理解 [3D 建模基础概念](basic-concepts.md)
    - 有制作 VRChat 服装的意愿
    - 对 Marvelous Designer 有基本兴趣

## 🎯 制作流程全貌

### 5 个主要阶段

!!! info "VRChat 服装制作的完整流程"

    **阶段1: 企划与准备** (10-20%)
    - 构思创意、确定设计
    - 收集参考资料、进行技术评估
    - 准备工具、配置环境

    **阶段2: 设计与建模** (30-40%)
    - 在 Marvelous Designer 中制作
    - 版型制作与适配
    - 物理设置与模拟

    **阶段3: 纹理与材质** (20-30%)
    - 制作与编辑纹理
    - 设置与调整材质
    - 完成外观的最终润色

    **阶段4: Unity 集成与优化** (20-25%)
    - 集成到 Unity 项目
    - 配置 VRChat SDK
    - 进行性能优化

    **阶段5: 测试与部署** (5-10%)
    - 在 VRChat 中测试
    - 最终调整与修复问题
    - 上传与发布

## 📋 阶段1: 企划与准备阶段

### 确定设计概念

!!! example "企划阶段的任务"

    **创意构思**:
    - 明确想做的服装愿景
    - 决定目标用途（自用 / 分发）
    - 评估技术可行性

    **收集参考资料**:
    - 实物服装照片与插画
    - 类似的 3D 服装作品
    - 制作技法参考资料

    **技术准备**:
    - 确认所需软件
    - 准备模型文件
    - 整理作业环境

### 时间分配与计划

!!! tip "高效企划的技巧"

    **面向初学者的时间分配**:
    - 设计推敲: 2-3 小时
    - 收集参考资料: 1-2 小时
    - 技术准备: 1 小时
    - **合计: 4-6 小时**

    **计划要点**:
    - 设定现实可行的目标
    - 逐步提升难度
    - 预留缓冲时间

## 🎨 阶段2: 设计与建模阶段

### 在 Marvelous Designer 中制作

!!! example "建模阶段的工作流"

    **基础制作流程**:
    1. **导入模型**: [模型导入指南](../workflows/avatar-import.md)
    2. **利用现有服装**: [服装适配](../workflows/garment-fitting.md)
    3. **原创制作**: [新建服装制作](../workflows/new-garment-creation.md)

    **质量管理要点**:
    - 勤保存、勤备份
    - 分阶段检查质量
    - 出现问题时尽早处理

### 物理设置与模拟

!!! info "物理设置的重要性"

    **调整物理特性**:
    - 根据布料种类进行设置
    - 考虑在 VRChat 中的动作表现
    - 平衡性能与效果

    **模拟质量**:
    - 作业中使用轻量设置
    - 最终输出使用高质量设置
    - 逐步提升质量

| 服装复杂度 | 制作时间 | 学习与调整时间 | 总时间 |

| **标准（礼服等）** | 6-12 小时 | 4-8 小时 | 10-20 小时 |
| **复杂（多层服装等）** | 15-30 小时 | 10-20 小时 | 25-50 小时 |

## 🖼️ 阶段3: 纹理与材质阶段

### 制作与编辑纹理

!!! example "纹理作业流程"

    **基础纹理作业**:
    1. 检查并调整 UV Map
    2. 制作基础颜色
    3. 添加细节纹理
    4. 实现特殊效果

    **推荐工具**:
    - **Photoshop**: 功能强大的编辑工具
    - **GIMP**: 免费替代方案
    - **Substance Painter**: 面向 3D 制作
    - **Marvelous Designer 内置功能**: 基础编辑

### 材质设置

!!! info "面向 VRChat 的材质优化"

    **重要设置项**:
    - 选择 Shader
    - 压缩纹理
    - 调整绘制质量
    - 性能优化

    **质量与性能的平衡**:
    - 根据用途选择质量等级
    - 控制文件大小
    - 确保显示质量

## 🎮 阶段4: Unity 集成与优化阶段

### Unity 集成作业

!!! example "Unity 集成基础流程"

    **集成步骤**:
    1. [Unity 项目设置](../unity/project-setup.md)
    2. 从 Marvelous Designer 导出 FBX
    3. 在 Unity 中设置材质
    4. 配置 VRChat SDK

    **优化作业**:
    - 优化多边形数量
    - 调整纹理尺寸
    - 删除不必要的元素
    - 进行性能测试

### VRChat SDK 设置

!!! warning "VRChat 的限制事项"

    **性能限制**:
    - 多边形数量限制
    - 纹理限制
    - 材质数量限制
    - 文件大小限制

    **应对方式**:
    - 分阶段优化
    - 在限制内提升质量
    - 研究替代方案

## ✅ 阶段5: 测试与部署阶段

### 在 VRChat 中测试

!!! example "测试项目检查清单"

    **基础动作测试**:
    - [ ] 确认服装显示正常
    - [ ] 基础动作下不出现破绽
    - [ ] 与模型的相互作用正常
    - [ ] 从其他用户视角也能正常显示

    **详细测试**:
    - [ ] 各类表情与手势
    - [ ] 坐下、躺下等动作
    - [ ] 物理运算是否自然
    - [ ] 对性能的影响

### 最终调整与部署

!!! success "部署准备"

    **最终确认项**:
    - 确认全部功能正常
    - 性能测试通过
    - 完成外观最终调整
    - 准备文档与说明

    **上传与发布**:
    - 上传到 VRChat
    - 确认并调整设置
    - 视需要准备分发

## ⚡ 提升效率的要点

### 常见瓶颈

!!! warning "降低制作效率的因素"

    **技术类瓶颈**:
    - 因知识不足导致作业停滞
    - 因质量问题返工
    - 因优化不足触及限制

    **计划与管理类瓶颈**:
    - 首次目标设置得过于复杂
    - 参考资料不足
    - 备份不足造成作业损失

### 提高效率的技巧

!!! tip "提升制作效率的方法"

    **技术层面的效率提升**:
    - 利用并改造现有服装
    - 创建并复用模板
    - 识别可自动化的作业

    **管理层面的效率提升**:
    - 分阶段设定目标
    - 定期确认进度
    - 尽早发现并处理问题

## 📊 项目管理

### 进度管理方法

!!! example "高效的项目管理"

    **任务管理**:
    - 细化各阶段的具体任务
    - 明确优先级
    - 设定完成标准

    **质量管理**:
    - 分阶段做质量检查
    - 记录问题与对策
    - 持续改进

### 文件与版本管理

!!! info "文件管理最佳实践"

    **推荐文件夹结构**:
    ```
    VRChat服装项目/
    ├── 01_企划/
    │   ├── 设计方案/
    │   └── 参考资料/
    ├── 02_Marvelous_Designer/
    │   ├── 项目文件/
    │   └── 导出文件/
    ├── 03_纹理/
    │   ├── 工作文件/
    │   └── 完成品/
    ├── 04_Unity/
    │   └── Assets/
    └── 05_完成品/
        └── 分发用/
    ```

    **版本管理**:
    - 定期备份
    - 保存重要里程碑
    - 记录变更历史
